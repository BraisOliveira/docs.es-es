### <a name="to-make-a-compile-time-reference-to-a-strong-named-assembly"></a><span data-ttu-id="d1689-109">Para hacer referencia en tiempo de compilación a un ensamblado con nombre seguro</span><span class="sxs-lookup"><span data-stu-id="d1689-109">To make a compile-time reference to a strong-named assembly</span></span>
