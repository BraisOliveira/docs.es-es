 <span data-ttu-id="17ce9-164">La solución consiste en pasar un búfer <xref:System.Text.StringBuilder> como argumento en lugar de una cadena.</span><span class="sxs-lookup"><span data-stu-id="17ce9-164">The solution is to pass a <xref:System.Text.StringBuilder> buffer as the argument instead of a string.</span></span> <span data-ttu-id="17ce9-165">El destinatario puede desreferenciar y modificar un `StringBuilder`, siempre que no exceda la capacidad del `StringBuilder`.</span><span class="sxs-lookup"><span data-stu-id="17ce9-165">A `StringBuilder` can be dereferenced and modified by the callee, provided it does not exceed the capacity of the `StringBuilder`.</span></span> <span data-ttu-id="17ce9-166">También puede inicializarse con una longitud fija.</span><span class="sxs-lookup"><span data-stu-id="17ce9-166">It can also be initialized to a fixed length.</span></span> <span data-ttu-id="17ce9-167">Por ejemplo, si inicializa un búfer `StringBuilder` con una capacidad de `N`, el contador de referencias proporcionará un búfer con un tamaño de (`N`+ 1) caracteres.</span><span class="sxs-lookup"><span data-stu-id="17ce9-167">For example, if you initialize a `StringBuilder` buffer to a capacity of `N`, the marshaler provides a buffer of size (`N`+1) characters.</span></span> <span data-ttu-id="17ce9-168">+ 1 tiene en cuenta el hecho de que la cadena no administrada tiene un terminador NULL mientras que `StringBuilder` no.</span><span class="sxs-lookup"><span data-stu-id="17ce9-168">The +1 accounts for the fact that the unmanaged string has a null terminator while `StringBuilder` does not.</span></span>  
  
 <span data-ttu-id="17ce9-169">Por ejemplo, la función `GetWindowText` de la API de Microsoft Windows (definida en Windows.h) es un búfer de caracteres de longitud fija que debe pasarse a código no administrado para su manipulación.</span><span class="sxs-lookup"><span data-stu-id="17ce9-169">For example, the Microsoft Windows API `GetWindowText` function (defined in Windows.h) is a fixed-length character buffer that must be passed into unmanaged code to be manipulated.</span></span> <span data-ttu-id="17ce9-170">`LpString` apunta a un búfer asignado por el llamador de tamaño `nMaxCount`.</span><span class="sxs-lookup"><span data-stu-id="17ce9-170">`LpString` points to a caller-allocated buffer of size `nMaxCount`.</span></span> <span data-ttu-id="17ce9-171">Se espera que el llamador asigne el búfer y establezca el argumento `nMaxCount` en el tamaño del búfer asignado.</span><span class="sxs-lookup"><span data-stu-id="17ce9-171">The caller is expected to allocate the buffer and set the `nMaxCount` argument to the size of the allocated buffer.</span></span> <span data-ttu-id="17ce9-172">El siguiente código muestra la declaración de función `GetWindowText`, tal y como se define en Windows.h.</span><span class="sxs-lookup"><span data-stu-id="17ce9-172">The following code shows the `GetWindowText` function declaration as defined in Windows.h.</span></span>  
  
```  
int GetWindowText(  
HWND hWnd,        // Handle to window or control.  
LPTStr lpString,  // Text buffer.  
int nMaxCount     // Maximum number of characters to copy.  
);  
```  
  
 <span data-ttu-id="17ce9-173">El destinatario puede desreferenciar y modificar un `StringBuilder`, siempre que no exceda la capacidad del `StringBuilder`.</span><span class="sxs-lookup"><span data-stu-id="17ce9-173">A `StringBuilder` can be dereferenced and modified by the callee, provided it does not exceed the capacity of the `StringBuilder`.</span></span> <span data-ttu-id="17ce9-174">En el ejemplo de código siguiente se muestra cómo inicializar `StringBuilder` con una longitud fija.</span><span class="sxs-lookup"><span data-stu-id="17ce9-174">The following code example demonstrates how `StringBuilder` can be initialized to a fixed length.</span></span>  
  
```vb  
Public Class Win32API  
Public Declare Auto Sub GetWindowText Lib "User32.Dll" _  
(h As Integer, s As StringBuilder, nMaxCount As Integer)  
End Class  
Public Class Window  
   Friend h As Integer ' Friend handle to Window.  
   Public Function GetText() As String  
      Dim sb As New StringBuilder(256)  
      Win32API.GetWindowText(h, sb, sb.Capacity + 1)  
   Return sb.ToString()  
   End Function  
End Class  
```  
  
```csharp  
public class Win32API {  
[DllImport("User32.Dll")]  
public static extern void GetWindowText(int h, StringBuilder s,   
int nMaxCount);  
}  
public class Window {  
   internal int h;        // Internal handle to Window.  
   public String GetText() {  
      StringBuilder sb = new StringBuilder(256);  
      Win32API.GetWindowText(h, sb, sb.Capacity + 1);  
   return sb.ToString();  
   }  
}  
```
