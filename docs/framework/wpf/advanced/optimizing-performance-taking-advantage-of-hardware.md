### <a name="graphics-rendering-tiers"></a><span data-ttu-id="8df8b-118">Niveles de representación de gráficos</span><span class="sxs-lookup"><span data-stu-id="8df8b-118">Graphics Rendering Tiers</span></span>

 <span data-ttu-id="8df8b-121">Para ello, [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] proporciona funcionalidad para determinar la funcionalidad de gráficos de un sistema en tiempo de ejecución.</span><span class="sxs-lookup"><span data-stu-id="8df8b-121">To achieve this, [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] provides functionality to determine the graphics capability of a system at runtime.</span></span> <span data-ttu-id="8df8b-122">La capacidad de gráficos se determina mediante la categorización de la tarjeta de vídeo como uno de tres niveles de capacidad de representación.</span><span class="sxs-lookup"><span data-stu-id="8df8b-122">Graphics capability is determined by categorizing the video card as one of three rendering capability tiers.</span></span> [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]<span data-ttu-id="8df8b-123">expone una API que permite que una aplicación consulte el nivel de capacidad de representación.</span><span class="sxs-lookup"><span data-stu-id="8df8b-123">exposes an API that allows an application to query the rendering capability tier.</span></span> <span data-ttu-id="8df8b-124">A continuación, la aplicación puede tomar diferentes rutas de código en tiempo de ejecución según el nivel de representación admitido por el hardware.</span><span class="sxs-lookup"><span data-stu-id="8df8b-124">Your application can then take different code paths at run time depending on the rendering tier supported by the hardware.</span></span>  
  
 <span data-ttu-id="8df8b-125">Las características del hardware gráfico que más afectan a los niveles de representación son:</span><span class="sxs-lookup"><span data-stu-id="8df8b-125">The features of the graphics hardware that most impact the rendering tier levels are:</span></span>  
  
- <span data-ttu-id="8df8b-126">**RAM de vídeo** La cantidad de memoria de vídeo en el hardware gráfico determina el tamaño y el número de búferes que puede utilizarse para la composición de gráficos.</span><span class="sxs-lookup"><span data-stu-id="8df8b-126">**Video RAM** The amount of video memory on the graphics hardware determines the size and number of buffers that can be used for compositing graphics.</span></span>  
  
- <span data-ttu-id="8df8b-127">**Sombreador de píxeles** Un sombreador de píxeles es una función de procesamiento de gráficos que calcula los efectos píxel por píxel.</span><span class="sxs-lookup"><span data-stu-id="8df8b-127">**Pixel Shader** A pixel shader is a graphics processing function that calculates effects on a per-pixel basis.</span></span> <span data-ttu-id="8df8b-128">Según la resolución de los gráficos mostrados, puede haber varios millones de píxeles que deban procesarse en cada fotograma mostrado.</span><span class="sxs-lookup"><span data-stu-id="8df8b-128">Depending on the resolution of the displayed graphics, there could be several million pixels that need to be processed for each display frame.</span></span>  
  
- <span data-ttu-id="8df8b-129">**Sombreador de vértices** Un sombreador de vértices es una función de procesamiento de gráficos que realiza operaciones matemáticas en los datos de vértice del objeto.</span><span class="sxs-lookup"><span data-stu-id="8df8b-129">**Vertex Shader** A vertex shader is a graphics processing function that performs mathematical operations on the vertex data of the object.</span></span>  
  
- <span data-ttu-id="8df8b-130">**Compatibilidad con texturas múltiples** La compatibilidad con texturas múltiples hace referencia a la capacidad de aplicar dos o más texturas distintas durante una operación de combinación en un objeto gráfico 3D.</span><span class="sxs-lookup"><span data-stu-id="8df8b-130">**Multitexture Support** Multitexture support refers to the ability to apply two or more distinct textures during a blending operation on a 3D graphics object.</span></span> <span data-ttu-id="8df8b-131">El grado de compatibilidad con texturas múltiples queda determinado por el número de unidades de múltiples texturas en el hardware gráfico.</span><span class="sxs-lookup"><span data-stu-id="8df8b-131">The degree of multitexture support is determined by the number of multitexture units on the graphics hardware.</span></span>  
  
 <span data-ttu-id="8df8b-132">El sombreador de píxeles, el sombreador de vértices y las características de varias texturas se utilizan para definir niveles de versión específicos de DirectX, que, a su vez, se usan [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]para definir los distintos niveles de representación en.</span><span class="sxs-lookup"><span data-stu-id="8df8b-132">The pixel shader, vertex shader, and multitexture features are used to define specific DirectX version levels, which, in turn, are used to define the different rendering tiers in [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].</span></span>

- <span data-ttu-id="8df8b-135">**Nivel de representación 0** Sin aceleración de hardware gráfico.</span><span class="sxs-lookup"><span data-stu-id="8df8b-135">**Rendering Tier 0** No graphics hardware acceleration.</span></span> <span data-ttu-id="8df8b-136">El nivel de versión de DirectX es anterior a la versión 7,0.</span><span class="sxs-lookup"><span data-stu-id="8df8b-136">The DirectX version level is less than version 7.0.</span></span>  
  
- <span data-ttu-id="8df8b-137">**Nivel de representación 1** Aceleración de hardware de gráficos parcial.</span><span class="sxs-lookup"><span data-stu-id="8df8b-137">**Rendering Tier 1** Partial graphics hardware acceleration.</span></span> <span data-ttu-id="8df8b-138">El nivel de versión de DirectX es mayor o igual que la versión 7,0 y **menor** que la versión 9,0.</span><span class="sxs-lookup"><span data-stu-id="8df8b-138">The DirectX version level is greater than or equal to version 7.0, and **lesser** than version 9.0.</span></span>  
  
- <span data-ttu-id="8df8b-139">**Nivel de representación 2** La mayoría de las características de gráficos utiliza la aceleración de hardware gráfico.</span><span class="sxs-lookup"><span data-stu-id="8df8b-139">**Rendering Tier 2** Most graphics features use graphics hardware acceleration.</span></span> <span data-ttu-id="8df8b-140">El nivel de versión de DirectX es mayor o igual que la versión 9,0.</span><span class="sxs-lookup"><span data-stu-id="8df8b-140">The DirectX version level is greater than or equal to version 9.0.</span></span>  
  
 <span data-ttu-id="8df8b-141">Para obtener más información [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] sobre la representación de niveles, vea [niveles de representación de gráficos](graphics-rendering-tiers.md).</span><span class="sxs-lookup"><span data-stu-id="8df8b-141">For more information on [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] rendering tiers, see [Graphics Rendering Tiers](graphics-rendering-tiers.md).</span></span>
