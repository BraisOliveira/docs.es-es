### <a name="graphics-rendering-tiers"></a><span data-ttu-id="021e3-118">Niveles de representación de gráficos</span><span class="sxs-lookup"><span data-stu-id="021e3-118">Graphics Rendering Tiers</span></span>

 <span data-ttu-id="021e3-121">Para lograrlo, [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] proporciona funcionalidad para determinar la capacidad gráfica de un sistema en tiempo de ejecución.</span><span class="sxs-lookup"><span data-stu-id="021e3-121">To achieve this, [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] provides functionality to determine the graphics capability of a system at runtime.</span></span> <span data-ttu-id="021e3-122">Capacidad para gráficos viene determinada por la clasificación de la tarjeta de vídeo como uno de los tres niveles de funcionalidad de representación.</span><span class="sxs-lookup"><span data-stu-id="021e3-122">Graphics capability is determined by categorizing the video card as one of three rendering capability tiers.</span></span> [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] <span data-ttu-id="021e3-123">expone una API que permite que una aplicación consultar el nivel de funcionalidad de representación.</span><span class="sxs-lookup"><span data-stu-id="021e3-123">exposes an API that allows an application to query the rendering capability tier.</span></span> <span data-ttu-id="021e3-124">La aplicación, a continuación, puede tomar diferentes rutas de código en tiempo de ejecución según el nivel de representación compatible con el hardware.</span><span class="sxs-lookup"><span data-stu-id="021e3-124">Your application can then take different code paths at run time depending on the rendering tier supported by the hardware.</span></span>  
  
 <span data-ttu-id="021e3-125">Las características del hardware gráfico que más afectan a los niveles de representación son:</span><span class="sxs-lookup"><span data-stu-id="021e3-125">The features of the graphics hardware that most impact the rendering tier levels are:</span></span>

- <span data-ttu-id="021e3-129">**Sombreador de vértices** Un sombreador de vértices es una función de procesamiento de gráficos que realiza operaciones matemáticas en los datos de vértice del objeto.</span><span class="sxs-lookup"><span data-stu-id="021e3-129">**Vertex Shader** A vertex shader is a graphics processing function that performs mathematical operations on the vertex data of the object.</span></span>  
  
- <span data-ttu-id="021e3-130">**Compatibilidad con texturas múltiples** La compatibilidad con texturas múltiples hace referencia a la capacidad de aplicar dos o más texturas distintas durante una operación de combinación en un objeto gráfico 3D.</span><span class="sxs-lookup"><span data-stu-id="021e3-130">**Multitexture Support** Multitexture support refers to the ability to apply two or more distinct textures during a blending operation on a 3D graphics object.</span></span> <span data-ttu-id="021e3-131">El grado de compatibilidad con texturas múltiples queda determinado por el número de unidades de múltiples texturas en el hardware gráfico.</span><span class="sxs-lookup"><span data-stu-id="021e3-131">The degree of multitexture support is determined by the number of multitexture units on the graphics hardware.</span></span>  
  
 <span data-ttu-id="021e3-132">El sombreador de píxeles, del sombreador de vértices y texturas múltiples características que se usan para definir específico [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] niveles de versión, que, a su vez, se usan para definir los niveles de representación diferente en [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].</span><span class="sxs-lookup"><span data-stu-id="021e3-132">The pixel shader, vertex shader, and multitexture features are used to define specific [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version levels, which, in turn, are used to define the different rendering tiers in [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].</span></span>

- <span data-ttu-id="021e3-135">**Nivel de representación 0** Sin aceleración de hardware gráfico.</span><span class="sxs-lookup"><span data-stu-id="021e3-135">**Rendering Tier 0** No graphics hardware acceleration.</span></span> <span data-ttu-id="021e3-136">El [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] el nivel de versión es menor que la versión 7.0.</span><span class="sxs-lookup"><span data-stu-id="021e3-136">The [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version level is less than version 7.0.</span></span>  
  
- <span data-ttu-id="021e3-137">**Nivel de representación 1** aceleración de hardware gráfico parcial.</span><span class="sxs-lookup"><span data-stu-id="021e3-137">**Rendering Tier 1** Partial graphics hardware acceleration.</span></span> <span data-ttu-id="021e3-138">El [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] nivel de versión es mayor o igual a la versión 7.0, y **menor** que la versión 9.0.</span><span class="sxs-lookup"><span data-stu-id="021e3-138">The [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version level is greater than or equal to version 7.0, and **lesser** than version 9.0.</span></span>  
  
- <span data-ttu-id="021e3-139">**Nivel de representación 2** La mayoría de las características de gráficos utiliza la aceleración de hardware gráfico.</span><span class="sxs-lookup"><span data-stu-id="021e3-139">**Rendering Tier 2** Most graphics features use graphics hardware acceleration.</span></span> <span data-ttu-id="021e3-140">El nivel de versión de [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] es mayor o igual que la versión 9.0.</span><span class="sxs-lookup"><span data-stu-id="021e3-140">The [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version level is greater than or equal to version 9.0.</span></span>
