- <span data-ttu-id="d82d9-120">Tenga en cuenta que en el comportamiento predeterminado de las acciones de compilación marcado, puede dejar la derivación en blanco en la definición de clase parcial en el lado de código subyacente.</span><span class="sxs-lookup"><span data-stu-id="d82d9-120">Note that under the default behavior of the markup compile build actions, you can leave the derivation blank in the partial class definition on the code-behind side.</span></span> <span data-ttu-id="d82d9-121">El resultado compilado supondrá que tipo de respaldo de la raíz de la página para ser la base de la clase parcial, incluso si no ha especificado.</span><span class="sxs-lookup"><span data-stu-id="d82d9-121">The compiled result will assume the page root's backing type to be the basis for the partial class, even if it not specified.</span></span> <span data-ttu-id="d82d9-122">Sin embargo, confiar en este comportamiento no es una práctica recomendada.</span><span class="sxs-lookup"><span data-stu-id="d82d9-122">However, relying on this behavior is not a best practice.</span></span>

- <span data-ttu-id="d82d9-126">El controlador debe coincidir con el delegado para el evento correspondiente en el sistema de tipos de respaldo.</span><span class="sxs-lookup"><span data-stu-id="d82d9-126">The handler must match the delegate for the appropriate event in the backing type system.</span></span>
