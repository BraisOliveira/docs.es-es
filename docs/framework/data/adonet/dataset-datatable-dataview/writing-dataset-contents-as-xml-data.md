# <a name="writing-dataset-contents-as-xml-data"></a><span data-ttu-id="61e64-102">Escribir el contenido de un conjunto de datos como datos XML</span><span class="sxs-lookup"><span data-stu-id="61e64-102">Writing DataSet Contents as XML Data</span></span>
<span data-ttu-id="61e64-103">En ADO.NET es posible escribir una representación XML de un <xref:System.Data.DataSet>, con o sin su esquema.</span><span class="sxs-lookup"><span data-stu-id="61e64-103">In ADO.NET you can write an XML representation of a <xref:System.Data.DataSet>, with or without its schema.</span></span> <span data-ttu-id="61e64-104">Si la información de esquema está incluida alineada con el código XML, se escribirá con el lenguaje de definición de esquemas XML (XSD).</span><span class="sxs-lookup"><span data-stu-id="61e64-104">If schema information is included inline with the XML, it is written using the XML Schema definition language (XSD).</span></span> <span data-ttu-id="61e64-105">El esquema contiene las definiciones de tabla del <xref:System.Data.DataSet>, así como las definiciones de relaciones y restricciones.</span><span class="sxs-lookup"><span data-stu-id="61e64-105">The schema contains the table definitions of the <xref:System.Data.DataSet> as well as the relation and constraint definitions.</span></span>  
  
 <span data-ttu-id="61e64-106">Cuando un <xref:System.Data.DataSet> se escribe como datos XML, las filas del <xref:System.Data.DataSet> se escriben en sus versiones actuales.</span><span class="sxs-lookup"><span data-stu-id="61e64-106">When a <xref:System.Data.DataSet> is written as XML data, the rows in the <xref:System.Data.DataSet> are written in their current versions.</span></span> <span data-ttu-id="61e64-107">Sin embargo, el <xref:System.Data.DataSet> también se puede escribir como un DiffGram, de forma que se incluyan los valores actuales y originales de las filas.</span><span class="sxs-lookup"><span data-stu-id="61e64-107">However, the <xref:System.Data.DataSet> can also be written as a DiffGram so that both the current and the original values of the rows will be included.</span></span>

```vb

```  
  
```csharp  
string xmlDS = custDS.GetXml();  
```  
  
 <span data-ttu-id="61e64-111">**GetXml** devuelve la representación XML de la <xref:System.Data.DataSet> sin información del esquema.</span><span class="sxs-lookup"><span data-stu-id="61e64-111">**GetXml** returns the XML representation of the <xref:System.Data.DataSet> without schema information.</span></span> <span data-ttu-id="61e64-112">Para escribir la información de esquema desde el <xref:System.Data.DataSet> (como esquema XML) en una cadena, utilice **GetXmlSchema**.</span><span class="sxs-lookup"><span data-stu-id="61e64-112">To write the schema information from the <xref:System.Data.DataSet> (as XML Schema) to a string, use **GetXmlSchema**.</span></span>

 <span data-ttu-id="61e64-117">En la tabla siguiente se muestra las opciones para **XmlWriteMode**.</span><span class="sxs-lookup"><span data-stu-id="61e64-117">The following table shows the options for **XmlWriteMode**.</span></span>  
  
|<span data-ttu-id="61e64-118">Opción XmlWriteMode</span><span class="sxs-lookup"><span data-stu-id="61e64-118">XmlWriteMode option</span></span>|<span data-ttu-id="61e64-119">Descripción</span><span class="sxs-lookup"><span data-stu-id="61e64-119">Description</span></span>|

 <span data-ttu-id="61e64-128">Al escribir una representación XML de un <xref:System.Data.DataSet> que contiene **DataRelation** objetos, probablemente le interesará el código XML resultante tenga las filas secundarias de cada relación anidadas dentro de sus elementos primarios relacionados.</span><span class="sxs-lookup"><span data-stu-id="61e64-128">When writing an XML representation of a <xref:System.Data.DataSet> that contains **DataRelation** objects, you will most likely want the resulting XML to have the child rows of each relation nested within their related parent elements.</span></span> <span data-ttu-id="61e64-129">Para ello, establezca el **Nested** propiedad de la **DataRelation** a **true** al agregar el **DataRelation** a la <xref:System.Data.DataSet>.</span><span class="sxs-lookup"><span data-stu-id="61e64-129">To accomplish this, set the **Nested** property of the **DataRelation** to **true** when you add the **DataRelation** to the <xref:System.Data.DataSet>.</span></span> <span data-ttu-id="61e64-130">Para obtener más información, consulte [anidar objetos DataRelation](../../../../../docs/framework/data/adonet/dataset-datatable-dataview/nesting-datarelations.md).</span><span class="sxs-lookup"><span data-stu-id="61e64-130">For more information, see [Nesting DataRelations](../../../../../docs/framework/data/adonet/dataset-datatable-dataview/nesting-datarelations.md).</span></span>

