-   <span data-ttu-id="28455-112">El extremo dependiente de la restricción.</span><span class="sxs-lookup"><span data-stu-id="28455-112">The dependent end of the constraint.</span></span> <span data-ttu-id="28455-113">Es un tipo de entidad que tiene una o varias propiedades que hacen referencia a la clave de entidad del extremo principal.</span><span class="sxs-lookup"><span data-stu-id="28455-113">(An entity type that has a property or properties that reference the entity key of the principal end.)</span></span>
