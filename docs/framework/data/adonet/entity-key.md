# <a name="entity-key"></a><span data-ttu-id="f5d27-102">clave de entidad</span><span class="sxs-lookup"><span data-stu-id="f5d27-102">entity key</span></span>
<span data-ttu-id="f5d27-103">Un *clave de entidad* es un [propiedad](../../../../docs/framework/data/adonet/property.md) o un conjunto de propiedades de un [tipo de entidad](../../../../docs/framework/data/adonet/entity-type.md) que se usan para determinar la identidad.</span><span class="sxs-lookup"><span data-stu-id="f5d27-103">An *entity key* is a [property](../../../../docs/framework/data/adonet/property.md) or a set of properties of an [entity type](../../../../docs/framework/data/adonet/entity-type.md) that are used to determine identity.</span></span> <span data-ttu-id="f5d27-104">Las propiedades que constituyen una entidad se eligen en tiempo de diseño.</span><span class="sxs-lookup"><span data-stu-id="f5d27-104">The properties that make up an entity key are chosen at design time.</span></span> <span data-ttu-id="f5d27-105">Los valores de las propiedades de clave de entidad deben identificar de forma exclusiva una instancia del tipo de entidad dentro de un [conjunto de entidades](../../../../docs/framework/data/adonet/entity-set.md) en tiempo de ejecución.</span><span class="sxs-lookup"><span data-stu-id="f5d27-105">The values of entity key properties must uniquely identify an entity type instance within an [entity set](../../../../docs/framework/data/adonet/entity-set.md) at run time.</span></span> <span data-ttu-id="f5d27-106">Las propiedades que constituyen una clave de entidad se deben elegir de tal forma que garanticen la unicidad de las instancias de un conjunto de entidades.</span><span class="sxs-lookup"><span data-stu-id="f5d27-106">The properties that make up an entity key should be chosen to guarantee uniqueness of instances in an entity set.</span></span>  
  
 <span data-ttu-id="f5d27-107">Los requisitos para que un conjunto de propiedades sea una clave de entidad son los siguientes:</span><span class="sxs-lookup"><span data-stu-id="f5d27-107">The following are the requirements for a set of properties to be an entity key:</span></span>  
  
-   <span data-ttu-id="f5d27-108">No puede haber dos claves de entidad idénticas en un conjunto de entidades.</span><span class="sxs-lookup"><span data-stu-id="f5d27-108">No two entity keys within an entity set can be identical.</span></span> <span data-ttu-id="f5d27-109">Es decir, para dos entidades cualesquiera dentro de un conjunto de entidades, los valores de todas las propiedades que constituyen una clave no pueden ser idénticos.</span><span class="sxs-lookup"><span data-stu-id="f5d27-109">That is, for any two entities within an entity set, the values for all of the properties that constitute a key cannot be the same.</span></span> <span data-ttu-id="f5d27-110">Sin embargo, algunos (pero no todos) los valores que constituyen una clave de entidad pueden ser idénticos.</span><span class="sxs-lookup"><span data-stu-id="f5d27-110">However, some (but not all) of the values that make up an entity key can be the same.</span></span>  
  
-   <span data-ttu-id="f5d27-111">Una clave de entidad debe constar de un conjunto de admitir valores NULL, inmutable, [propiedades de tipo primitivo](../../../../docs/framework/data/adonet/entity-data-model-primitive-data-types.md).</span><span class="sxs-lookup"><span data-stu-id="f5d27-111">An entity key must consist of a set of non-nullable, immutable, [primitive type properties](../../../../docs/framework/data/adonet/entity-data-model-primitive-data-types.md).</span></span>  
  
-   <span data-ttu-id="f5d27-112">Las propiedades que constituyen una clave de entidad para un tipo de entidad determinado no pueden cambiar.</span><span class="sxs-lookup"><span data-stu-id="f5d27-112">The properties that make up an entity key for a given entity type cannot change.</span></span> <span data-ttu-id="f5d27-113">No se puede designar más de una clave de entidad posible para un tipo de entidad determinado; no se admiten las claves suplentes.</span><span class="sxs-lookup"><span data-stu-id="f5d27-113">You cannot allow more than one possible entity key for a given entity type; surrogate keys are not supported.</span></span>  
  
-   <span data-ttu-id="f5d27-114">Cuando una entidad está implicada en una jerarquía de herencia, la entidad raíz debe contener todas las propiedades que constituyen la clave de entidad, y esta se debe definir en el tipo de entidad raíz.</span><span class="sxs-lookup"><span data-stu-id="f5d27-114">When an entity is involved in an inheritance hierarchy, the root entity must contain all the properties that make up the entity key, and the entity key must be defined on the root entity type.</span></span> <span data-ttu-id="f5d27-115">Para obtener más información, consulte [Entity Data Model: Herencia](../../../../docs/framework/data/adonet/entity-data-model-inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="f5d27-115">For more information, see [Entity Data Model: Inheritance](../../../../docs/framework/data/adonet/entity-data-model-inheritance.md).</span></span>
