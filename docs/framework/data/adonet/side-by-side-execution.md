## <a name="sqlcommand-execution"></a><span data-ttu-id="e603d-135">Ejecución de SqlCommand</span><span class="sxs-lookup"><span data-stu-id="e603d-135">SqlCommand Execution</span></span>


 <span data-ttu-id="e603d-145">Este cambio puede afectar a la compatibilidad de versiones anteriores y de la aplicación si depende del comportamiento de <xref:System.Data.SqlClient.SqlCommand.ExecuteReader%2A> para cualquier versión de .NET Framework.</span><span class="sxs-lookup"><span data-stu-id="e603d-145">This change can affect both the forward and backward compatibility of your application if you depend on the behavior of <xref:System.Data.SqlClient.SqlCommand.ExecuteReader%2A> for either version of the .NET Framework.</span></span>
