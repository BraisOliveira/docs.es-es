## <a name="remarks"></a><span data-ttu-id="d460e-129">Comentarios</span><span class="sxs-lookup"><span data-stu-id="d460e-129">Remarks</span></span>

<span data-ttu-id="d460e-130">En .NET Framework 3.5 y versiones anteriores, si carga un ensamblado desde una ubicación remota, el código en el ensamblado se ejecuta en confianza parcial con un conjunto de permisos que depende de la zona desde la que se carga.</span><span class="sxs-lookup"><span data-stu-id="d460e-130">In the .NET Framework 3.5 and earlier versions, if you load an assembly from a remote location, code in the assembly runs in partial trust with a grant set that depends on the zone from which it is loaded.</span></span> <span data-ttu-id="d460e-131">Por ejemplo, si carga un ensamblado desde un sitio Web, se cargan en la zona de Internet y a conceder el conjunto de permisos de Internet.</span><span class="sxs-lookup"><span data-stu-id="d460e-131">For example, if you load an assembly from a website, it is loaded into the Internet zone and granted the Internet permission set.</span></span> <span data-ttu-id="d460e-132">En otras palabras, ejecuta en un espacio aislado de Internet.</span><span class="sxs-lookup"><span data-stu-id="d460e-132">In other words, it executes in an Internet sandbox.</span></span>

- <span data-ttu-id="d460e-139">Ejecutar código de ensamblado de plena confianza.</span><span class="sxs-lookup"><span data-stu-id="d460e-139">Run the assembly's code in full trust.</span></span> <span data-ttu-id="d460e-140">Ello, configure el `<loadFromRemoteSources>` elemento.</span><span class="sxs-lookup"><span data-stu-id="d460e-140">You do this by configuring the `<loadFromRemoteSources>` element.</span></span> <span data-ttu-id="d460e-141">Le permite especificar que los ensamblados que se ejecutan en confianza parcial en versiones anteriores de .NET Framework ahora se ejecutan con plena confianza en .NET Framework 4 y versiones posteriores.</span><span class="sxs-lookup"><span data-stu-id="d460e-141">It lets you specify that the assemblies that run in partial trust in earlier versions of the .NET Framework now run in full trust in the .NET Framework 4 and later versions.</span></span>
