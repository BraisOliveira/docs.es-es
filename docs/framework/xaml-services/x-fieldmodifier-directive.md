# <a name="xfieldmodifier-directive"></a><span data-ttu-id="b73dc-102">x:FieldModifier (Directiva)</span><span class="sxs-lookup"><span data-stu-id="b73dc-102">x:FieldModifier Directive</span></span>
<span data-ttu-id="b73dc-103">Modifica el comportamiento de compilación de XAML para que se definen los campos para las referencias a objetos con nombre con <xref:System.Reflection.TypeAttributes.Public?displayProperty=nameWithType> acceso en lugar de la <xref:System.Reflection.TypeAttributes.NotPublic?displayProperty=nameWithType> comportamiento predeterminado.</span><span class="sxs-lookup"><span data-stu-id="b73dc-103">Modifies XAML compilation behavior so that fields for named object references are defined with <xref:System.Reflection.TypeAttributes.Public?displayProperty=nameWithType> access instead of the <xref:System.Reflection.TypeAttributes.NotPublic?displayProperty=nameWithType> default behavior.</span></span>  
  
## <a name="xaml-attribute-usage"></a><span data-ttu-id="b73dc-104">Uso de atributos XAML</span><span class="sxs-lookup"><span data-stu-id="b73dc-104">XAML Attribute Usage</span></span>  
  
```xaml  
<object x:FieldModifier="Public".../>  
```
