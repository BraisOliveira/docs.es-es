### <a name="to-author-a-composite-control"></a><span data-ttu-id="04239-111">Para crear un control compuesto</span><span class="sxs-lookup"><span data-stu-id="04239-111">To author a composite control</span></span>  
  
1.  <span data-ttu-id="04239-112">Abra un nuevo proyecto **Aplicación Windows** denominado `DemoControlHost`.</span><span class="sxs-lookup"><span data-stu-id="04239-112">Open a new **Windows Application** project called `DemoControlHost`.</span></span>  
  
2.  <span data-ttu-id="04239-113">En el menú **Proyecto** , haga clic en **Agregar control de usuario**.</span><span class="sxs-lookup"><span data-stu-id="04239-113">On the **Project** menu, click **Add User Control**.</span></span>  
  
3.  <span data-ttu-id="04239-114">En el cuadro de diálogo **Agregar nuevo elemento**, dé al archivo de clase (archivo .vb o .cs) el nombre que desea para el control compuesto.</span><span class="sxs-lookup"><span data-stu-id="04239-114">In the **Add New Item** dialog box, give the class file (.vb or .cs file) the name you want the composite control to have.</span></span>  
  
4.  <span data-ttu-id="04239-115">Seleccione el **agregar** botón para crear el archivo de clase para el control compuesto.</span><span class="sxs-lookup"><span data-stu-id="04239-115">Select the **Add** button to create the class file for the composite control.</span></span>  
  
5.  <span data-ttu-id="04239-116">Agregue controles del **Cuadro de herramientas** a la superficie del control compuesto.</span><span class="sxs-lookup"><span data-stu-id="04239-116">Add controls from the **Toolbox** to the composite control surface.</span></span>  
  
6.  <span data-ttu-id="04239-117">Escriba código en los procedimientos de evento para controlar los eventos que produzca el control compuesto o sus controles constituyentes.</span><span class="sxs-lookup"><span data-stu-id="04239-117">Place code in event procedures, to handle events raised by the composite control or by its constituent controls.</span></span>  
  
7.  <span data-ttu-id="04239-118">Cierre el diseñador para el control compuesto y guarde el archivo cuando se le indique.</span><span class="sxs-lookup"><span data-stu-id="04239-118">Close the designer for the composite control, and save the file when you are prompted.</span></span>  
  
8.  <span data-ttu-id="04239-119">En el menú **Compilar** , haga clic en **Compilar solución**.</span><span class="sxs-lookup"><span data-stu-id="04239-119">On the **Build** menu, click **Build Solution**.</span></span>  
  
     <span data-ttu-id="04239-120">Para que los controles personalizados aparezcan en el **Cuadro de herramientas**, deberá compilar el proyecto.</span><span class="sxs-lookup"><span data-stu-id="04239-120">The project must be built in order for custom controls to appear in the **Toolbox**.</span></span>  
  
9. <span data-ttu-id="04239-121">Use la pestaña **DemoControlHost** del **Cuadro de herramientas** para agregar instancias del control a `Form1`.</span><span class="sxs-lookup"><span data-stu-id="04239-121">Use the **DemoControlHost** tab of the **Toolbox** to add instances of your control to `Form1`.</span></span>  
  
### <a name="to-author-a-control-class-library"></a><span data-ttu-id="04239-122">Para crear una biblioteca de clases de controles</span><span class="sxs-lookup"><span data-stu-id="04239-122">To author a control class library</span></span>  
  
1.  <span data-ttu-id="04239-123">Abra un nuevo proyecto de tipo **Biblioteca de controles de Windows**.</span><span class="sxs-lookup"><span data-stu-id="04239-123">Open a new **Windows Control Library** project.</span></span>  
  
     <span data-ttu-id="04239-124">De forma predeterminada, el proyecto contiene un control compuesto.</span><span class="sxs-lookup"><span data-stu-id="04239-124">By default, the project contains a composite control.</span></span>  
  
2.  <span data-ttu-id="04239-125">Agregue controles y código como se describió en el procedimiento anterior.</span><span class="sxs-lookup"><span data-stu-id="04239-125">Add controls and code as described in the procedure above.</span></span>  
  
3.  <span data-ttu-id="04239-126">Elija un control cuyas clases de herencia no desee que cambien y establezca la propiedad **Modifiers** de este control en **Private**.</span><span class="sxs-lookup"><span data-stu-id="04239-126">Choose a control you do not want inheriting classes to change, and set the **Modifiers** property of this control to **Private**.</span></span>  
  
4.  <span data-ttu-id="04239-127">Compile el archivo DLL.</span><span class="sxs-lookup"><span data-stu-id="04239-127">Build the DLL.</span></span>  
  
### <a name="to-inherit-from-a-composite-control-in-a-control-class-library"></a><span data-ttu-id="04239-128">Para heredar de un control compuesto de una biblioteca de clases de controles</span><span class="sxs-lookup"><span data-stu-id="04239-128">To inherit from a composite control in a control class library</span></span>  
  
1.  <span data-ttu-id="04239-129">En el menú **Archivo**, seleccione **Agregar** y **Nuevo proyecto** para agregar un nuevo proyecto **Aplicación Windows** a la solución.</span><span class="sxs-lookup"><span data-stu-id="04239-129">On the **File** menu, point to **Add** and select **New Project** to add a new **Windows Application** project to the solution.</span></span>  
  
2.  <span data-ttu-id="04239-130">En el **Explorador de soluciones**, haga clic con el botón derecho en la carpeta **Referencias** del nuevo proyecto y elija **Agregar referencia** para abrir el cuadro de diálogo **Agregar referencia**.</span><span class="sxs-lookup"><span data-stu-id="04239-130">In **Solution Explorer**, right-click the **References** folder for the new project and choose **Add Reference** to open the **Add Reference** dialog box.</span></span>  
  
3.  <span data-ttu-id="04239-131">Seleccione la pestaña **Proyectos** y haga doble clic en el proyecto de Biblioteca de controles.</span><span class="sxs-lookup"><span data-stu-id="04239-131">Select the **Projects** tab and double-click your control library project.</span></span>  
  
4.  <span data-ttu-id="04239-132">En el menú **Compilar** , haga clic en **Compilar solución**.</span><span class="sxs-lookup"><span data-stu-id="04239-132">On the **Build** menu, click **Build Solution**.</span></span>  
  
5.  <span data-ttu-id="04239-133">En el **Explorador de soluciones**, haga clic con el botón derecho en el proyecto de Biblioteca de controles y elija **Agregar nuevo elemento** en el menú contextual.</span><span class="sxs-lookup"><span data-stu-id="04239-133">In **Solution Explorer**, right-click your control library project and select **Add New Item** from the shortcut menu.</span></span>  
  
6.  <span data-ttu-id="04239-134">Seleccione la plantilla **Control de usuario heredado** en el cuadro de diálogo **Agregar nuevo elemento**.</span><span class="sxs-lookup"><span data-stu-id="04239-134">Select the **Inherited User Control** template from the **Add New Item** dialog box.</span></span>  
  
7.  <span data-ttu-id="04239-135">En el cuadro de diálogo **Selector de herencia**, haga doble clic en el control del que desea heredar.</span><span class="sxs-lookup"><span data-stu-id="04239-135">In the **Inheritance Picker** dialog box, double-click the control you want to inherit from.</span></span>  
  
     <span data-ttu-id="04239-136">Se agrega un nuevo control al proyecto.</span><span class="sxs-lookup"><span data-stu-id="04239-136">A new control is added to your project.</span></span>  
  
8.  <span data-ttu-id="04239-137">Abra el diseñador visual para el nuevo control y agregue controles constituyentes adicionales.</span><span class="sxs-lookup"><span data-stu-id="04239-137">Open the visual designer for the new control and add additional constituent controls.</span></span>  
  
     <span data-ttu-id="04239-138">Puede ver los controles constituyentes heredados del control de usuario en la DLL, así como alterar las propiedades de aquellos controles cuya propiedad **Modifiers** sea **Public**.</span><span class="sxs-lookup"><span data-stu-id="04239-138">You can see the constituent controls that were inherited from the composite control in your DLL, and you can alter the properties of controls whose **Modifiers** property is **Public**.</span></span> <span data-ttu-id="04239-139">En cambio, no puede cambiar las propiedades del control cuya propiedad **Modifiers** sea **Private**.</span><span class="sxs-lookup"><span data-stu-id="04239-139">You cannot change the properties of the control whose **Modifiers** property is **Private**.</span></span>
