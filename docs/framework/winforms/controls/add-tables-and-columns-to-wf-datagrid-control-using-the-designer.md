### <a name="to-add-a-table-to-the-datagrid-control-in-the-designer"></a><span data-ttu-id="3a2a5-116">Para agregar una tabla al control DataGrid en el diseñador</span><span class="sxs-lookup"><span data-stu-id="3a2a5-116">To add a table to the DataGrid control in the designer</span></span>

2.  <span data-ttu-id="3a2a5-119">Seleccione el <xref:System.Windows.Forms.DataGrid> del control <xref:System.Windows.Forms.DataGrid.TableStyles%2A> propiedad en la ventana Propiedades y, a continuación, haga clic en el botón de puntos suspensivos (![de pantalla de VisualStudioEllipsesButton](../media/vbellipsesbutton.png "vbEllipsesButton")) junto a la propiedad para mostrar el **Editor de colección DataGridTableStyle**.</span><span class="sxs-lookup"><span data-stu-id="3a2a5-119">Select the <xref:System.Windows.Forms.DataGrid> control's <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property in the Properties window, and then click the ellipsis button (![VisualStudioEllipsesButton screenshot](../media/vbellipsesbutton.png "vbEllipsesButton")) next to the property to display the **DataGridTableStyle Collection Editor**.</span></span>  
  
3.  <span data-ttu-id="3a2a5-120">En el editor de colecciones, haga clic en **agregar** para insertar un estilo de tabla.</span><span class="sxs-lookup"><span data-stu-id="3a2a5-120">In the collection editor, click **Add** to insert a table style.</span></span>

5.  <span data-ttu-id="3a2a5-123">En el **miembros** cuadro del editor de la colección, haga clic en el estilo de tabla.</span><span class="sxs-lookup"><span data-stu-id="3a2a5-123">In the **Members** box of the collection editor, click the table style.</span></span>  
  
6.  <span data-ttu-id="3a2a5-124">En el **propiedades** cuadro del editor de colección, seleccione la <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> valor de la tabla que desea mostrar.</span><span class="sxs-lookup"><span data-stu-id="3a2a5-124">In the **Properties** box of the collection editor, select the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> value for the table you want to display.</span></span>
