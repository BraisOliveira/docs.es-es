### <a name="to-change-the-value-of-an-existing-setting-at-design-time-in-c"></a><span data-ttu-id="776c5-104">Para cambiar el valor de una configuración existente en tiempo de diseño en C\#</span><span class="sxs-lookup"><span data-stu-id="776c5-104">To Change the Value of an Existing Setting at Design Time in C\#</span></span>
