## <a name="findresponse"></a><span data-ttu-id="d1f6a-140">FindResponse</span><span class="sxs-lookup"><span data-stu-id="d1f6a-140">FindResponse</span></span>  
 <xref:System.ServiceModel.Discovery.FindResponse> <span data-ttu-id="d1f6a-141">tiene un <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> propiedad de colección que contiene todas las respuestas enviadas por los servicios en la red coincidentes.</span><span class="sxs-lookup"><span data-stu-id="d1f6a-141">has an <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> collection property that contains any replies sent by matching services on the network.</span></span> <span data-ttu-id="d1f6a-142">Si ningún servicio responde, la colección está vacía.</span><span class="sxs-lookup"><span data-stu-id="d1f6a-142">If no services replied, the collection is empty.</span></span> <span data-ttu-id="d1f6a-143">Si uno o más servicios respondieran, cada respuesta se almacenaría en un objeto <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata>, que contiene la dirección, el contrato y cierta información adicional sobre el servicio.</span><span class="sxs-lookup"><span data-stu-id="d1f6a-143">If one or more services replied, each reply is stored in an <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> object, which contains the address, contract, and some additional information about the service.</span></span>  
  
 <span data-ttu-id="d1f6a-144">En el siguiente ejemplo, se muestra cómo realizar una operación de búsqueda en código.</span><span class="sxs-lookup"><span data-stu-id="d1f6a-144">The following example shows how to perform a find operation in code.</span></span>  
  
```  
// Create DiscoveryClient  
DiscoveryClient discoveryClient = new DiscoveryClient(new UdpDiscoveryEndpoint());  
  
// Create FindCriteria  
FindCriteria findCriteria = new FindCriteria(typeof(IPrinterService));  
findCriteria.Scopes.Add(new Uri("http://www.contoso.com/building1/floor1"));  
findCriteria.Duration = TimeSpan.FromSeconds(10);   
  
// Find ICalculatorService endpoints              
FindResponse findResponse = discoveryClient.Find(findCriteria);  
  
Console.WriteLine("Found {0} ICalculatorService endpoint(s).", findResponse.Endpoints.Count)  
```
