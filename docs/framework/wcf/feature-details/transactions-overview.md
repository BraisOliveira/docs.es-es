# <a name="windows-communication-foundation-transactions-overview"></a><span data-ttu-id="f00aa-102">Información general sobre las transacciones de Windows Communication Foundation</span><span class="sxs-lookup"><span data-stu-id="f00aa-102">Windows Communication Foundation Transactions Overview</span></span>
<span data-ttu-id="f00aa-103">Las transacciones proporcionan una manera de agrupar un conjunto de acciones u operaciones en una unidad indivisible única de ejecución.</span><span class="sxs-lookup"><span data-stu-id="f00aa-103">Transactions provide a way to group a set of actions or operations into a single indivisible unit of execution.</span></span> <span data-ttu-id="f00aa-104">Una transacción es una colección de operaciones con las propiedades siguientes:</span><span class="sxs-lookup"><span data-stu-id="f00aa-104">A transaction is a collection of operations with the following properties:</span></span>  
  
-   <span data-ttu-id="f00aa-105">Atomicidad.</span><span class="sxs-lookup"><span data-stu-id="f00aa-105">Atomicity.</span></span> <span data-ttu-id="f00aa-106">Esto garantiza que todas las actualizaciones completadas en una transacción concreta se confirmadas y sean duraderas o que sean anuladas y reviertan a su estado anterior.</span><span class="sxs-lookup"><span data-stu-id="f00aa-106">This ensures that either all of the updates completed under a specific transaction are committed and made durable or they are all aborted and rolled back to their previous state.</span></span>  
  
-   <span data-ttu-id="f00aa-107">Coherencia</span><span class="sxs-lookup"><span data-stu-id="f00aa-107">Consistency.</span></span> <span data-ttu-id="f00aa-108">Esto garantiza que los cambios realizados en una transacción representan una transformación desde un estado coherente a otro.</span><span class="sxs-lookup"><span data-stu-id="f00aa-108">This guarantees that the changes made under a transaction represent a transformation from one consistent state to another.</span></span> <span data-ttu-id="f00aa-109">Por ejemplo, una transacción que transfiere el dinero de una cuenta corriente a una cuenta de ahorros no cambia la cantidad de dinero en la cuenta bancaria general.</span><span class="sxs-lookup"><span data-stu-id="f00aa-109">For example, a transaction that transfers money from a checking account to a savings account does not change the amount of money in the overall bank account.</span></span>  
  
-   <span data-ttu-id="f00aa-110">Aislamiento.</span><span class="sxs-lookup"><span data-stu-id="f00aa-110">Isolation.</span></span> <span data-ttu-id="f00aa-111">Esto evita que una transacción obedezca a cambios no confirmados que pertenecen a otras transacciones simultáneas.</span><span class="sxs-lookup"><span data-stu-id="f00aa-111">This prevents a transaction from observing uncommitted changes belonging to other concurrent transactions.</span></span> <span data-ttu-id="f00aa-112">El aislamiento proporciona una abstracción de simultaneidad y garantiza, al mismo tiempo, que una transacción no tenga un impacto inesperado en la ejecución de otra transacción.</span><span class="sxs-lookup"><span data-stu-id="f00aa-112">Isolation provides an abstraction of concurrency while ensuring one transaction cannot have an unexpected impact on the execution of another transaction.</span></span>  
  
-   <span data-ttu-id="f00aa-113">Duración.</span><span class="sxs-lookup"><span data-stu-id="f00aa-113">Durability.</span></span> <span data-ttu-id="f00aa-114">Esto significa que una vez confirmado, las actualizaciones de recursos administrados (como un registro de base de datos) serán persistentes frente a errores.</span><span class="sxs-lookup"><span data-stu-id="f00aa-114">This means that once committed, updates to managed resources (such as a database record) will be persistent in the face of failures.</span></span>  
  
 <span data-ttu-id="f00aa-115">Windows Communication Foundation (WCF) proporciona un amplio conjunto de características que le permiten crear transacciones distribuidas en la aplicación de servicio Web.</span><span class="sxs-lookup"><span data-stu-id="f00aa-115">Windows Communication Foundation (WCF) provides a rich set of features that enable you to create distributed transactions in your Web service application.</span></span>  
  
 <span data-ttu-id="f00aa-116">WCF implementa la compatibilidad con el protocolo WS-AtomicTransaction (WS-AT) que permite a las aplicaciones WCF envíe transacciones a aplicaciones interoperables, como servicios Web interoperables compilados mediante tecnología de otro fabricante.</span><span class="sxs-lookup"><span data-stu-id="f00aa-116">WCF implements support for the WS-AtomicTransaction (WS-AT) protocol that enables WCF applications to flow transactions to interoperable applications, such as interoperable Web services built using third-party technology.</span></span> <span data-ttu-id="f00aa-117">WCF también implementa la compatibilidad con el protocolo de transacciones OLE, que puede usarse en escenarios donde no es necesario en función de interoperabilidad para habilitar el flujo de transacción.</span><span class="sxs-lookup"><span data-stu-id="f00aa-117">WCF also implements support for the OLE Transactions protocol, which can be used in scenarios where you do not need interop functionality to enable transaction flow.</span></span>  
  
 <span data-ttu-id="f00aa-118">Puede utilizar un archivo de configuración de la aplicación para configurar los enlaces con el fin de habilitar o deshabilitar el flujo de la transacción, así como establecer el protocolo de transacción deseado en un enlace.</span><span class="sxs-lookup"><span data-stu-id="f00aa-118">You can use an application configuration file to configure bindings to enable or disable transaction flow, as well as set the desired transaction protocol on a binding.</span></span> <span data-ttu-id="f00aa-119">Además, puede establecer los tiempos de espera de la transacción en el nivel del servicio utilizando el archivo de configuración.</span><span class="sxs-lookup"><span data-stu-id="f00aa-119">In addition, you can set transaction time-outs at the service level using the configuration file.</span></span> <span data-ttu-id="f00aa-120">Para obtener más información, consulte [Habilitar flujo de transacción](../../../../docs/framework/wcf/feature-details/enabling-transaction-flow.md).</span><span class="sxs-lookup"><span data-stu-id="f00aa-120">For more information, see [Enabling Transaction Flow](../../../../docs/framework/wcf/feature-details/enabling-transaction-flow.md).</span></span>  
  
 <span data-ttu-id="f00aa-121">Los atributos de transacción en el espacio de nombres <xref:System.ServiceModel> le permiten hacer lo siguiente:</span><span class="sxs-lookup"><span data-stu-id="f00aa-121">Transaction attributes in the <xref:System.ServiceModel> namespace allow you to do the following:</span></span>
