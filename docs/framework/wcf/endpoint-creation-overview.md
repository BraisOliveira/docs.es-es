## <a name="defining-endpoints-in-code"></a><span data-ttu-id="8eaf0-131">Definir los extremos en código</span><span class="sxs-lookup"><span data-stu-id="8eaf0-131">Defining Endpoints in Code</span></span>  
 <span data-ttu-id="8eaf0-132">El siguiente ejemplo muestra cómo especificar un extremo en código:</span><span class="sxs-lookup"><span data-stu-id="8eaf0-132">The following example illustrates how to specify an endpoint in code with the following:</span></span>  
  
-   <span data-ttu-id="8eaf0-133">Defina un contrato para un `IEcho` tipo de servicio que acepta el nombre de alguien y eco con la respuesta "Hello \<nombre >!".</span><span class="sxs-lookup"><span data-stu-id="8eaf0-133">Define a contract for an `IEcho` type of service that accepts someone's name and echo with the response "Hello \<name>!".</span></span>  
  
-   <span data-ttu-id="8eaf0-134">Implemente un servicio `Echo` del tipo definido por el contrato `IEcho`.</span><span class="sxs-lookup"><span data-stu-id="8eaf0-134">Implement an `Echo` service of the type defined by the `IEcho` contract.</span></span>  
  
-   <span data-ttu-id="8eaf0-135">Especifique una dirección de punto de conexión de `http://localhost:8000/Echo` para el servicio.</span><span class="sxs-lookup"><span data-stu-id="8eaf0-135">Specify an endpoint address of `http://localhost:8000/Echo` for the service.</span></span>  
  
-   <span data-ttu-id="8eaf0-136">Configure el servicio `Echo` mediante un enlace <xref:System.ServiceModel.WSHttpBinding>.</span><span class="sxs-lookup"><span data-stu-id="8eaf0-136">Configure the `Echo` service using a <xref:System.ServiceModel.WSHttpBinding> binding.</span></span>  
  
```csharp  
Namespace Echo  
{  
   // Define the contract for the IEcho service   
   [ServiceContract]  
   public interface IEcho  
   {  
       [OperationContract]  
       String Hello(string name)  
   }  
  
   // Create an Echo service that implements IEcho contract  
   class Echo : IEcho  
   {  
      public string Hello(string name)  
      {  
         return "Hello" + name + "!";  
      }  
      public static void Main ()  
      {  
          //Specify the base address for Echo service.  
          Uri echoUri = new Uri("http://localhost:8000/");  
  
          //Create a ServiceHost for the Echo service.  
          ServiceHost serviceHost = new ServiceHost(typeof(Echo),echoUri);  
  
          // Use a predefined WSHttpBinding to configure the service.  
          WSHttpBinding binding = new WSHttpBinding();  
  
          // Add the endpoint for this service to the service host.  
          serviceHost.AddServiceEndpoint(  
             typeof(IEcho),   
             binding,   
             echoUri  
           );  
  
          // Open the service host to run it.  
          serviceHost.Open();  
     }  
  }  
}  
```
