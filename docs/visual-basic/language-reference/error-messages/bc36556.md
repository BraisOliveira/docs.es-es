# <a name="anonymous-type-member-name-can-be-inferred-only-from-a-simple-or-qualified-name-with-no-arguments"></a><span data-ttu-id="710c2-102">El nombre de miembro de tipo anónimo sólo se puede inferir a partir de un nombre simple o completo sin argumentos</span><span class="sxs-lookup"><span data-stu-id="710c2-102">Anonymous type member name can be inferred only from a simple or qualified name with no arguments</span></span>

## <a name="example"></a><span data-ttu-id="710c2-105">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="710c2-105">Example</span></span>
