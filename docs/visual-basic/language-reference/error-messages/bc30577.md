# <a name="addressof-operand-must-be-the-name-of-a-method-without-parentheses"></a><span data-ttu-id="4a40f-102">El operando 'AddressOf' debe ser el nombre de un método (sin paréntesis)</span><span class="sxs-lookup"><span data-stu-id="4a40f-102">'AddressOf' operand must be the name of a method (without parentheses)</span></span>

<span data-ttu-id="4a40f-103">El operador `AddressOf` crea una instancia de delegado de procedimiento que hace referencia a un procedimiento específico.</span><span class="sxs-lookup"><span data-stu-id="4a40f-103">The `AddressOf` operator creates a procedure delegate instance that references a specific procedure.</span></span> <span data-ttu-id="4a40f-104">La sintaxis es la siguiente:</span><span class="sxs-lookup"><span data-stu-id="4a40f-104">The syntax is as follows:</span></span>

```vb
AddressOf procedurename
```

<span data-ttu-id="4a40f-105">Se han insertado paréntesis alrededor del argumento siguiente a `AddressOf`, donde no se necesita ninguno.</span><span class="sxs-lookup"><span data-stu-id="4a40f-105">You inserted parentheses around the argument following `AddressOf`, where none are needed.</span></span>

<span data-ttu-id="4a40f-106">**Identificador de error:** BC30577</span><span class="sxs-lookup"><span data-stu-id="4a40f-106">**Error ID:** BC30577</span></span>

## <a name="example"></a><span data-ttu-id="4a40f-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="4a40f-107">Example</span></span>

<span data-ttu-id="4a40f-108">En el ejemplo siguiente se genera bc30577:</span><span class="sxs-lookup"><span data-stu-id="4a40f-108">The following example generates bc30577:</span></span>

```vb
Public Sub CountZeroToTen()
    For i = 0 To 10
        Console.WriteLine($"Counted: {i}")
        Threading.Thread.Sleep(500)
    Next
End Sub

Sub Main()
    ' Any of the following two lines generates bc30577.
    'Dim t As New Threading.Thread(AddressOf(CountZeroToTen))
    'Dim t As New Threading.Thread(AddressOf CountZeroToTen())
    t.Start()
End Sub
```

## <a name="to-correct-this-error"></a><span data-ttu-id="4a40f-109">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="4a40f-109">To correct this error</span></span>

1. <span data-ttu-id="4a40f-110">Quite los paréntesis alrededor del argumento siguiente `AddressOf` como se muestra en el ejemplo siguiente:</span><span class="sxs-lookup"><span data-stu-id="4a40f-110">Remove the parentheses around the argument following `AddressOf` as shown in the following example:</span></span>

    ```vb
    Public Sub CountZeroToTen()
        For i = 0 To 10
            Console.WriteLine($"Counted: {i}")
            Threading.Thread.Sleep(500)
        Next
    End Sub

    Sub Main()
        Dim t As New Threading.Thread(AddressOf CountZeroToTen)
        t.Start()
    End Sub
    ```

2. <span data-ttu-id="4a40f-111">Asegúrese de que el argumento es un nombre de método.</span><span class="sxs-lookup"><span data-stu-id="4a40f-111">Make sure the argument is a method name.</span></span>
