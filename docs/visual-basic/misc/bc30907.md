<span data-ttu-id="567b2-103">Una estructura de herencia da como resultado una dependencia circular entre clases anidadas, es decir, dos clases que heredan entre sí.</span><span class="sxs-lookup"><span data-stu-id="567b2-103">An inheritance structure results in circular dependency among nested classes, that is, two classes inheriting from each other.</span></span>  
  
 <span data-ttu-id="567b2-104">El código siguiente puede generar este mensaje de error.</span><span class="sxs-lookup"><span data-stu-id="567b2-104">The following code can generate this error message.</span></span>  
  
```

    Inherits c3.c4  
    Public Class c2  
    End Class  
End Class  
Public Class c3  
    Inherits c1.c2  
    Public Class c4  
    End Class  
End Class  
```  
  
 <span data-ttu-id="567b2-105">En el código anterior, la clase `c1` hereda de la clase `c4`, pero `c4` está anidada dentro de `c3`, que hereda de `c2`, anidada dentro de `c1`.</span><span class="sxs-lookup"><span data-stu-id="567b2-105">In the preceding code, class `c1` inherits from class `c4`, but `c4` is nested inside `c3`, which inherits from `c2`, nested inside `c1`.</span></span>
