# <a name="continue-must-be-followed-by-do-for-or-while"></a><span data-ttu-id="56ed5-102">La instrucción 'Continue' debe ir seguida de 'Do', 'For' o 'While'</span><span class="sxs-lookup"><span data-stu-id="56ed5-102">'Continue' must be followed by 'Do', 'For' or 'While'</span></span>
<span data-ttu-id="56ed5-103">Una instrucción `Continue` debe ir seguida de `Do`, `For`o `While`, según si la instrucción `Continue` aparece dentro de un bucle `Do...Loop` , `For...Next` o `While...End While` .</span><span class="sxs-lookup"><span data-stu-id="56ed5-103">A `Continue` statement must be followed by `Do`, `For`, or `While`, depending on if the `Continue` statement appears within a `Do...Loop` loop, `For...Next` loop, or `While...End While` loop.</span></span>  
  
 <span data-ttu-id="56ed5-104">**Identificador de error:** BC30781</span><span class="sxs-lookup"><span data-stu-id="56ed5-104">**Error ID:** BC30781</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="56ed5-105">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="56ed5-105">To correct this error</span></span>  
  
1. <span data-ttu-id="56ed5-106">Si la instrucción `Continue` está en un bucle `Do...Loop` , cambie la instrucción a `Continue Do`.</span><span class="sxs-lookup"><span data-stu-id="56ed5-106">If the `Continue` statement is in a `Do...Loop` loop, change the statement to `Continue Do`.</span></span>  
  
2. <span data-ttu-id="56ed5-107">Si la instrucción `Continue` está en un bucle `For...Next` , cambie la instrucción a `Continue For`.</span><span class="sxs-lookup"><span data-stu-id="56ed5-107">If the `Continue` statement is in a `For...Next` loop, change the statement to `Continue For`.</span></span>  
  
3. <span data-ttu-id="56ed5-108">Si la instrucción `Continue` está en un bucle `While...End While` , cambie la instrucción a `Continue While`.</span><span class="sxs-lookup"><span data-stu-id="56ed5-108">If the `Continue` statement is in a `While...End While` loop, change the statement to `Continue While`.</span></span>  
  
4. <span data-ttu-id="56ed5-109">En otras circunstancias, quite la instrucción `Continue` .</span><span class="sxs-lookup"><span data-stu-id="56ed5-109">Otherwise, remove the `Continue` statement.</span></span>
