# <a name="continue-must-be-followed-by-do-for-or-while"></a><span data-ttu-id="e2296-102">La instrucción 'Continue' debe ir seguida de 'Do', 'For' o 'While'</span><span class="sxs-lookup"><span data-stu-id="e2296-102">'Continue' must be followed by 'Do', 'For' or 'While'</span></span>
<span data-ttu-id="e2296-103">Una instrucción `Continue` debe ir seguida de `Do`, `For`o `While`, según si la instrucción `Continue` aparece dentro de un bucle `Do...Loop` , `For...Next` o `While...End While` .</span><span class="sxs-lookup"><span data-stu-id="e2296-103">A `Continue` statement must be followed by `Do`, `For`, or `While`, depending on if the `Continue` statement appears within a `Do...Loop` loop, `For...Next` loop, or `While...End While` loop.</span></span>  
  
 <span data-ttu-id="e2296-104">**Identificador de error:** BC30781</span><span class="sxs-lookup"><span data-stu-id="e2296-104">**Error ID:** BC30781</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e2296-105">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="e2296-105">To correct this error</span></span>
