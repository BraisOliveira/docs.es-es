## <a name="example"></a><span data-ttu-id="afc9d-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="afc9d-109">Example</span></span>  
 <span data-ttu-id="afc9d-110">La clase siguiente contiene una propiedad predeterminada, `defaultProp`, que requiere un argumento entero.</span><span class="sxs-lookup"><span data-stu-id="afc9d-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="afc9d-111">Dado que la propiedad predeterminada requiere un argumento, la siguiente declaración genera un error.</span><span class="sxs-lookup"><span data-stu-id="afc9d-111">Because the default property requires an argument, the following declaration causes an error.</span></span>  
  
```

```
