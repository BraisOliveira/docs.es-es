# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="e6a20-102">El parámetro 'ByRef' '\<parametername >' no se puede usar en una expresión lambda</span><span class="sxs-lookup"><span data-stu-id="e6a20-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="e6a20-103">Una expresión lambda se declarada dentro de una declaración `Sub` o una función no puede usar cualquier parámetros `ByRef` de dicha `Sub` o función.</span><span class="sxs-lookup"><span data-stu-id="e6a20-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="e6a20-104">Por ejemplo, el siguiente código provocará este error porque el parámetro `ByRef` de `n` se usa en la expresión lambda.</span><span class="sxs-lookup"><span data-stu-id="e6a20-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="e6a20-105">**Identificador de error:** BC36639</span><span class="sxs-lookup"><span data-stu-id="e6a20-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e6a20-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="e6a20-106">To correct this error</span></span>

    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
