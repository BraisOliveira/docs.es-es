# <a name="extension-method-methodname-defined-in-modulename-is-not-generic-or-has-no-free-type-parameters-and-so-cannot-have-type-arguments"></a><span data-ttu-id="ed5a5-102">Método de extensión '\<methodname >' definido en '\<NombreMódulo >' no es genérico (o no tiene ningún parámetro de tipo libre) y por lo tanto, no puede tener argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="ed5a5-102">Extension method '\<methodname>' defined in '\<modulename>' is not generic (or has no free type parameters) and so cannot have type arguments</span></span>
<span data-ttu-id="ed5a5-103">Se ha especificado un argumento de tipo en una llamada a un método de extensión que no tiene parámetros genéricos o no tiene parámetros genéricos para los que aún no se ha especificado un tipo.</span><span class="sxs-lookup"><span data-stu-id="ed5a5-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="ed5a5-104">Por ejemplo, el código siguiente causa este error.</span><span class="sxs-lookup"><span data-stu-id="ed5a5-104">For example, the following code causes this error.</span></span>  
  
```vb  
' The extension method is not generic.  
<Extension()> _  
Sub Example(ByVal str As String)  
    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```
