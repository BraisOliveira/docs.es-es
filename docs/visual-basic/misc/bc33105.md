# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="c6f62-102">Los operandos 'If' no pueden ser argumentos con nombre</span><span class="sxs-lookup"><span data-stu-id="c6f62-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="c6f62-103">No se pueden usar argumentos con nombre en los operandos del operador `If` .</span><span class="sxs-lookup"><span data-stu-id="c6f62-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="c6f62-104">El ejemplo siguiente causa este error:</span><span class="sxs-lookup"><span data-stu-id="c6f62-104">The following example causes this error:</span></span>  
  
```vb

Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="c6f62-105">Esto difiere de la función `IIf` , que permite argumentos con nombre, como se muestra en el código siguiente:</span><span class="sxs-lookup"><span data-stu-id="c6f62-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```vb

```  
  
 <span data-ttu-id="c6f62-106">**IDENTIFICADOR de error:** BC33105</span><span class="sxs-lookup"><span data-stu-id="c6f62-106">**Error ID:** BC33105</span></span>
