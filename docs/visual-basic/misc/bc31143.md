# <a name="method-methodname-does-not-have-a-signature-compatible-with-delegate-delegatename"></a><span data-ttu-id="afc13-102">Método '\<methodname >' no tiene una firma compatible con el delegado \<'nombredelegado' ></span><span class="sxs-lookup"><span data-stu-id="afc13-102">Method '\<methodname>' does not have a signature compatible with delegate \<'delegatename'></span></span>
<span data-ttu-id="afc13-103">Este error se produce cuando se requiere una conversión entre un método y un delegado que no es posible.</span><span class="sxs-lookup"><span data-stu-id="afc13-103">This error occurs when a conversion is required between a method and a delegate that is not possible.</span></span> <span data-ttu-id="afc13-104">La causa del error puede ser una conversión entre parámetros o, cuando el método y el delegado son funciones, la conversión de los valores devueltos.</span><span class="sxs-lookup"><span data-stu-id="afc13-104">The cause of the error can be conversion between parameters or, when the method and delegate are functions, conversion in the return values.</span></span>  
  
 <span data-ttu-id="afc13-105">El código siguiente muestra el error en conversiones.</span><span class="sxs-lookup"><span data-stu-id="afc13-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="afc13-106">El delegado es `FunDel`.</span><span class="sxs-lookup"><span data-stu-id="afc13-106">The delegate is `FunDel`.</span></span>  
  
```vb  
Delegate Function FunDel(ByVal i As Integer, ByVal d As Double) As Integer  
```  
  
 <span data-ttu-id="afc13-107">Las funciones siguientes difieren de `FunDel` de manera que se producirá este error.</span><span class="sxs-lookup"><span data-stu-id="afc13-107">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>  
  
```vb  
Function ExampleMethod1(ByVal m As Integer, ByVal aDate As Date) As Integer  
End Function  
  
Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date  
End Function  
```  
  
 <span data-ttu-id="afc13-108">Cada una de las siguientes instrucciones de asignación produce el error.</span><span class="sxs-lookup"><span data-stu-id="afc13-108">Each of the following assignment statements causes the error.</span></span>  
  
```vb  
Sub Main()  
    ' The second parameters of FunDel and ExampleMethod1, Double and Date,  
    ' are not compatible.  
    'Dim d1 As FunDel = AddressOf ExampleMethod1  
  
    ' The return types of FunDel and ExampleMethod2, Integer and Date,  
    ' are not compatible.  
    'Dim d2 As FunDel = AddressOf ExampleMethod2  
  
End Sub  
```  
  
 <span data-ttu-id="afc13-109">**Identificador de error:** BC31143</span><span class="sxs-lookup"><span data-stu-id="afc13-109">**Error ID:** BC31143</span></span>
