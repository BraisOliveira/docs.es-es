# <a name="how-to-dispose-of-a-system-resource-visual-basic"></a><span data-ttu-id="05ea5-102">Filtrar Eliminar un recurso de sistema (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="05ea5-102">How to: Dispose of a System Resource (Visual Basic)</span></span>
<span data-ttu-id="05ea5-103">Puede usar un `Using` bloque para garantizar que el sistema elimina un recurso al que el código sale del bloque.</span><span class="sxs-lookup"><span data-stu-id="05ea5-103">You can use a `Using` block to guarantee that the system disposes of a resource when your code exits the block.</span></span> <span data-ttu-id="05ea5-104">Esto es útil si está utilizando un recurso del sistema que consume una gran cantidad de memoria o que otros componentes que desee utilizar.</span><span class="sxs-lookup"><span data-stu-id="05ea5-104">This is useful if you are using a system resource that consumes a large amount of memory, or that other components also want to use.</span></span>  
  
### <a name="to-dispose-of-a-database-connection-when-your-code-is-finished-with-it"></a><span data-ttu-id="05ea5-105">Para eliminar una conexión de base de datos cuando lo esté utilizando el código</span><span class="sxs-lookup"><span data-stu-id="05ea5-105">To dispose of a database connection when your code is finished with it</span></span>  
  
1.  <span data-ttu-id="05ea5-106">Asegúrese de incluir la correspondiente [instrucción Imports (tipo y Namespace. NET)](../../../../visual-basic/language-reference/statements/imports-statement-net-namespace-and-type.md) para la conexión de base de datos al principio del archivo de origen (en este caso, <xref:System.Data.SqlClient>).</span><span class="sxs-lookup"><span data-stu-id="05ea5-106">Make sure you include the appropriate [Imports Statement (.NET Namespace and Type)](../../../../visual-basic/language-reference/statements/imports-statement-net-namespace-and-type.md) for the database connection at the beginning of your source file (in this case, <xref:System.Data.SqlClient>).</span></span>  
  
2.  <span data-ttu-id="05ea5-107">Crear un `Using` bloquear con el `Using` y `End Using` instrucciones.</span><span class="sxs-lookup"><span data-stu-id="05ea5-107">Create a `Using` block with the `Using` and `End Using` statements.</span></span> <span data-ttu-id="05ea5-108">Dentro del bloque, coloque el código que se ocupa de la conexión de base de datos.</span><span class="sxs-lookup"><span data-stu-id="05ea5-108">Inside the block, put the code that deals with the database connection.</span></span>  
  
3.  <span data-ttu-id="05ea5-109">Declare la conexión y cree una instancia de ella como parte de la `Using` instrucción.</span><span class="sxs-lookup"><span data-stu-id="05ea5-109">Declare the connection and create an instance of it as part of the `Using` statement.</span></span>  
  
    ```  
    ' Insert the following line at the beginning of your source file.  
    Imports System.Data.SqlClient  
    Public Sub AccessSql(ByVal s As String)  
        Using sqc As New System.Data.SqlClient.SqlConnection(s)  
            MsgBox("Connected with string """ & sqc.ConnectionString & """")  
        End Using  
    End Sub  
    ```  
  
     <span data-ttu-id="05ea5-110">El sistema elimina el recurso independientemente de cómo salga del bloque, incluido el caso de una excepción no controlada.</span><span class="sxs-lookup"><span data-stu-id="05ea5-110">The system disposes of the resource no matter how you exit the block, including the case of an unhandled exception.</span></span>  
  
     <span data-ttu-id="05ea5-111">Tenga en cuenta que no se puede obtener acceso a `sqc` desde fuera de la `Using` block, porque su ámbito se limite al bloque.</span><span class="sxs-lookup"><span data-stu-id="05ea5-111">Note that you cannot access `sqc` from outside the `Using` block, because its scope is limited to the block.</span></span>  
  
     <span data-ttu-id="05ea5-112">Puede usar esta misma técnica en un recurso del sistema como un identificador de archivo o un contenedor COM.</span><span class="sxs-lookup"><span data-stu-id="05ea5-112">You can use this same technique on a system resource such as a file handle or a COM wrapper.</span></span> <span data-ttu-id="05ea5-113">Usa un `Using` bloquear cuando para asegurarse de que desea que deje el recurso disponible para otros componentes una vez que haya salido el `Using` bloque.</span><span class="sxs-lookup"><span data-stu-id="05ea5-113">You use a `Using` block when you want to be sure to leave the resource available for other components after you have exited the `Using` block.</span></span>
