## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="38188-106">Ejemplo de conversión ambigua</span><span class="sxs-lookup"><span data-stu-id="38188-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="38188-107">Este error puede producirse cuando no existe ninguna conversión o si las conversiones implícitas disponibles provocan una situación ambigua.</span><span class="sxs-lookup"><span data-stu-id="38188-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="38188-108">En el ejemplo siguiente se genera el error CS0151:</span><span class="sxs-lookup"><span data-stu-id="38188-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="38188-109">Ejemplo de un método void</span><span class="sxs-lookup"><span data-stu-id="38188-109">Example of void method</span></span>

<span data-ttu-id="38188-110">La invocación de un método [void](../../../csharp/language-reference/keywords/void.md) en una expresión de coincidencia [switch](../keywords/switch.md) genera el error CS0151.</span><span class="sxs-lookup"><span data-stu-id="38188-110">A [void](../../../csharp/language-reference/keywords/void.md) method invocation in a [switch](../keywords/switch.md) match expression generates CS0151.</span></span> <span data-ttu-id="38188-111">El error se puede corregir si en su lugar se llama a un método que devuelve un tipo entero como [int](../../../csharp/language-reference/builtin-types/integral-numeric-types.md) o [long](../../../csharp/language-reference/builtin-types/integral-numeric-types.md).</span><span class="sxs-lookup"><span data-stu-id="38188-111">You can fix the error by calling a method that returns an integral type such as [int](../../../csharp/language-reference/builtin-types/integral-numeric-types.md) or [long](../../../csharp/language-reference/builtin-types/integral-numeric-types.md) instead.</span></span>

```csharp

{
    static void Main()
    {
        switch (M()) // CS0151
        {
            default:
                break;
        }

    {
    }
}
```
