## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="00851-106">Ejemplo de conversión ambigua</span><span class="sxs-lookup"><span data-stu-id="00851-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="00851-107">Este error puede producirse cuando no existe ninguna conversión o si las conversiones implícitas disponibles provocan una situación ambigua.</span><span class="sxs-lookup"><span data-stu-id="00851-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="00851-108">En el ejemplo siguiente se genera el error CS0151:</span><span class="sxs-lookup"><span data-stu-id="00851-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="00851-109">Ejemplo de un método void</span><span class="sxs-lookup"><span data-stu-id="00851-109">Example of void method</span></span>

```csharp
class C
{
    static void Main()
    {
        switch (M()) // CS0151
        {
            default:
                break;
        }
    }

    static void M()
    {
    }
}
```
