 <span data-ttu-id="42814-106">Puede sobrecargar un operador para hacer que sea compatible con operandos de determinados tipos.</span><span class="sxs-lookup"><span data-stu-id="42814-106">You can overload an operator to make it support operands of certain types.</span></span> <span data-ttu-id="42814-107">Para obtener más información, vea [Sobrecarga de operadores](../../../csharp/language-reference/operators/operator-overloading.md).</span><span class="sxs-lookup"><span data-stu-id="42814-107">For more information, see [Operator overloading](../../../csharp/language-reference/operators/operator-overloading.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="42814-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="42814-108">Example</span></span>

 <span data-ttu-id="42814-109">En el ejemplo siguiente, CS0019 se genera en dos lugares porque [bool](../../../csharp/language-reference/keywords/bool.md) en C# no puede convertirse en [int](../../../csharp/language-reference/builtin-types/integral-numeric-types.md). CS0019 también se genera cuando el operador de resta se aplica a una cadena.</span><span class="sxs-lookup"><span data-stu-id="42814-109">In the following example, CS0019 is generated in two places because [bool](../../../csharp/language-reference/keywords/bool.md) in C# is not convertible to [int](../../../csharp/language-reference/builtin-types/integral-numeric-types.md). CS0019 also is generated when the subtraction operator is applied to a string.</span></span> <span data-ttu-id="42814-110">El operador de suma (+) puede usarse con operandos de cadena porque ese operador está sobrecargado mediante la clase `String` para realizar la concatenación de cadenas.</span><span class="sxs-lookup"><span data-stu-id="42814-110">The addition operator (+) can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>  

```csharp
static void Main()  
{  
    bool result = true;  
    if (result > 0) //CS0019  
    {  
        // Do something.  
    }  
  
    int i = 1;  
    // You cannot compare an integer and a boolean value.  
    if (i == true) //CS0019  
    {  
        //Do something...  
    }  
  
    // The following use of == causes no error. It is the comparison of
    // an integer and a boolean value that causes the error in the
    // previous if statement.  
    if (result == true)  
    {  
        //Do something...  
    }  
  
    string s = "Just try to subtract me.";  
    float f = 100 - s; // CS0019  
}  
```
