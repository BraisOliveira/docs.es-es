## <a name="example"></a><span data-ttu-id="d2c0f-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="d2c0f-106">Example</span></span>  
 <span data-ttu-id="d2c0f-107">En el ejemplo siguiente se genera el error CS0115.</span><span class="sxs-lookup"><span data-stu-id="d2c0f-107">The following sample generates CS0115.</span></span> <span data-ttu-id="d2c0f-108">Puede resolver el error CS0115 de dos maneras:</span><span class="sxs-lookup"><span data-stu-id="d2c0f-108">You can resolve CS0115 in one of two ways:</span></span>

- <span data-ttu-id="d2c0f-110">Use `MyClass1` como una clase base para `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="d2c0f-110">Use `MyClass1` as a base class for `MyClass2`.</span></span>  
  
```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
