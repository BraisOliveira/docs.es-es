# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="1f69f-102">Advertencia del compilador (nivel 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="1f69f-102">Compiler Warning (level 1) CS0420</span></span>
<span data-ttu-id="1f69f-103">'identifier': una referencia a un campo volátil no se tratará como tal</span><span class="sxs-lookup"><span data-stu-id="1f69f-103">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="1f69f-104">Normalmente, no se debe pasar un campo volatile mediante un parámetro `ref` u **out**, puesto que no se tratará como volátil dentro del ámbito de la función.</span><span class="sxs-lookup"><span data-stu-id="1f69f-104">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="1f69f-105">Existen excepciones a esta norma, como cuando se llama llamar a una API entrelazada.</span><span class="sxs-lookup"><span data-stu-id="1f69f-105">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="1f69f-106">Como con cualquier advertencia, puede usar [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) para deshabilitar esta advertencia en los infrecuentes casos en que se use un campo volatile de manera intencionada como parámetro de referencia.</span><span class="sxs-lookup"><span data-stu-id="1f69f-106">As with any warning, you may use the [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="1f69f-107">En el ejemplo siguiente se genera la advertencia CS0420:</span><span class="sxs-lookup"><span data-stu-id="1f69f-107">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420   
   }  
}  
```
