# <a name="compiler-error-cs1942"></a><span data-ttu-id="eef9d-102">Error del compilador CS1942</span><span class="sxs-lookup"><span data-stu-id="eef9d-102">Compiler Error CS1942</span></span>

<span data-ttu-id="eef9d-103">El tipo de la expresión de la cláusula 'clause' es incorrecto.</span><span class="sxs-lookup"><span data-stu-id="eef9d-103">The type of the expression in the 'clause' clause is incorrect.</span></span> <span data-ttu-id="eef9d-104">No se pudo realizar la inferencia de tipos en la llamada a 'method'.</span><span class="sxs-lookup"><span data-stu-id="eef9d-104">Type inference failed in the call to 'method'.</span></span>  
  
<span data-ttu-id="eef9d-105">Este error se genera normalmente cuando a la variable de rango se le ha proporcionado un tipo explícito incorrecto.</span><span class="sxs-lookup"><span data-stu-id="eef9d-105">This error is typically generated when the range variable has been given an incorrect explicit type.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="eef9d-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="eef9d-106">To correct this error</span></span>

1. <span data-ttu-id="eef9d-107">Si la variable de rango tiene un tipo explícito, asegúrese de que el tipo se pueda convertir de manera implícita del tipo de los elementos de la colección que recorre en iteración, o que sea igual que este.</span><span class="sxs-lookup"><span data-stu-id="eef9d-107">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="eef9d-108">Si la variable de rango está precedida por la palabra clave `var`, quite `var`.</span><span class="sxs-lookup"><span data-stu-id="eef9d-108">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="eef9d-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="eef9d-109">Example</span></span>

<span data-ttu-id="eef9d-110">El código siguiente genera el error CS1942:</span><span class="sxs-lookup"><span data-stu-id="eef9d-110">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="eef9d-111">CS1942 está relacionado con CS1949 porque el uso de `var` con una variable de rango hace que la operación `Cast<T>` subyacente produzca un error porque `var` no es un tipo.</span><span class="sxs-lookup"><span data-stu-id="eef9d-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
