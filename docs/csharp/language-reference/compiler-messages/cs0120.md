# <a name="compiler-error-cs0120"></a><span data-ttu-id="5d540-102">Error del compilador CS0120</span><span class="sxs-lookup"><span data-stu-id="5d540-102">Compiler Error CS0120</span></span>

<span data-ttu-id="5d540-103">Se requiere una referencia a objeto para el campo, método o propiedad no estáticos 'member'</span><span class="sxs-lookup"><span data-stu-id="5d540-103">An object reference is required for the nonstatic field, method, or property 'member'</span></span>

 <span data-ttu-id="5d540-104">Para usar un campo no estático, método o propiedad, primero debe crear una instancia de objeto.</span><span class="sxs-lookup"><span data-stu-id="5d540-104">In order to use a non-static field, method, or property, you must first create an object instance.</span></span> <span data-ttu-id="5d540-105">Para obtener más información sobre los métodos estáticos, vea [Clases estáticas y sus miembros](../../programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span><span class="sxs-lookup"><span data-stu-id="5d540-105">For more information about static methods, see [Static Classes and Static Class Members](../../programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span></span> <span data-ttu-id="5d540-106">Para obtener más información sobre la creación de instancias de clases, vea [Constructores de instancias](../../programming-guide/classes-and-structs/instance-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="5d540-106">For more information about creating instances of classes, see [Instance Constructors](../../programming-guide/classes-and-structs/instance-constructors.md).</span></span>

 <span data-ttu-id="5d540-107">En el ejemplo siguiente se genera el error CS0120:</span><span class="sxs-lookup"><span data-stu-id="5d540-107">The following sample generates CS0120:</span></span>

```csharp
// CS0120_1.cs
public class MyClass
{
    // Non-static field.
    public int i;
    // Non-static method.
    public void f() {}
    // Non-static property.
    int Prop
    {
        get
        {
            return 1;
        }
    }

    public static void Main()
    {
        i = 10;   // CS0120
        f();   // CS0120
        int p = Prop;   // CS0120
        // Try the following lines instead.
        // var mc = new MyClass();
        // mc.i = 10;
        // mc.f();
        // int p = mc.Prop;
    }
}
```

 <span data-ttu-id="5d540-108">CS0120 también se generará si existe una llamada a un método no estático desde un método estático, de la manera siguiente:</span><span class="sxs-lookup"><span data-stu-id="5d540-108">CS0120 will also be generated if there is a call to a non-static method from a static method, as follows:</span></span>

```csharp
// CS0120_2.cs
// CS0120 expected
using System;

public class MyClass
{
    public static void Main()  
    {  
        TestCall();   // CS0120
        // To call a non-static method from Main,
        // first create an instance of the class.
        // Use the following two lines instead:
        // var anInstanceofMyClass = new MyClass();
        // anInstanceofMyClass.TestCall();
   }

   public void TestCall()
   {
   }
}
```

 <span data-ttu-id="5d540-109">De manera similar, un método estático no puede llamar a un método de instancia a no ser que le proporcione explícitamente una instancia de la clase, de la manera siguiente:</span><span class="sxs-lookup"><span data-stu-id="5d540-109">Similarly, a static method cannot call an instance method unless you explicitly give it an instance of the class, as follows:</span></span>

```csharp
// CS0120_3.cs
using System;

public class MyClass
{
   public static void Main()
   {
      DoIt("Hello There");   // CS0120
   }
  
   private void DoIt(string sText)
   // You could also add the keyword static to the method definition.
   // private static void DoIt(string sText)
   {
      Console.WriteLine(sText);
   }
}
```
