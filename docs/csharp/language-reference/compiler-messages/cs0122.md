## <a name="example"></a><span data-ttu-id="9ed7c-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="9ed7c-108">Example</span></span>  
 <span data-ttu-id="9ed7c-109">En el ejemplo siguiente se genera el error CS0122:</span><span class="sxs-lookup"><span data-stu-id="9ed7c-109">The following sample generates CS0122:</span></span>  
  
```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}  
  
public class MyClass2  
{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```
