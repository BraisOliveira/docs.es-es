# <a name="compiler-error-cs0052"></a><span data-ttu-id="190f9-102">Error del compilador CS0052</span><span class="sxs-lookup"><span data-stu-id="190f9-102">Compiler Error CS0052</span></span>

<span data-ttu-id="190f9-103">Incoherencia de accesibilidad: el tipo de campo "type" es menos accesible que el campo "field"</span><span class="sxs-lookup"><span data-stu-id="190f9-103">Inconsistent accessibility: field type 'type' is less accessible than field 'field'</span></span>

 <span data-ttu-id="190f9-104">El tipo de un campo no puede ser menos accesible que el propio campo porque todas las construcciones públicas deben devolver un objeto accesible públicamente.</span><span class="sxs-lookup"><span data-stu-id="190f9-104">The type of a field cannot be less accessible than the field itself because all public constructs must return a publicly accessible object.</span></span>

## <a name="example"></a><span data-ttu-id="190f9-105">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="190f9-105">Example</span></span>

 <span data-ttu-id="190f9-106">En el ejemplo siguiente se genera el error CS0052:</span><span class="sxs-lookup"><span data-stu-id="190f9-106">The following sample generates CS0052:</span></span>

```csharp
// CS0052.cs
public class MyClass2
{
    // The following line causes an error because the field, M, is declared
    // as public, but the type, MyClass, is private. Therefore the type is
    // less accessible than the field.
    public MyClass M;   // CS0052

    private class MyClass
    {
    }
    // One way to resolve the error is to change the accessibility of the type
    // to public.
    //public class MyClass
    // Another solution is to change the accessibility of the field to private.
    // private MyClass M;
}

public class MainClass
{
    public static void Main()
    {
    }
}
```
