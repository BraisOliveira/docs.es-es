<span data-ttu-id="6d426-126">Use el operador [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) para obtener la instancia <xref:System.Type?displayProperty=nameWithType> que representa el tipo especificado:</span><span class="sxs-lookup"><span data-stu-id="6d426-126">Use the [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>
