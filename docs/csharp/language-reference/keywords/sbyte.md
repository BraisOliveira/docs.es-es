<span data-ttu-id="35eae-130">No se pueden convertir implícitamente los tipos numéricos no literales de mayor tamaño de almacenamiento a `sbyte` (vea en la [Tabla de tipos enteros](../../../csharp/language-reference/keywords/integral-types-table.md) los tamaños de almacenamiento de los tipos enteros).</span><span class="sxs-lookup"><span data-stu-id="35eae-130">You cannot implicitly convert nonliteral numeric types of larger storage size to `sbyte` (see [Integral Types Table](../../../csharp/language-reference/keywords/integral-types-table.md) for the storage sizes of integral types).</span></span> <span data-ttu-id="35eae-131">Considere, por ejemplo, las dos siguientes variables de `sbyte` `x` e `y`:</span><span class="sxs-lookup"><span data-stu-id="35eae-131">Consider, for example, the following two `sbyte` variables `x` and `y`:</span></span>
