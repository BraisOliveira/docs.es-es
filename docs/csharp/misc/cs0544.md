 <span data-ttu-id="507f8-104">Se intentó reemplazar un tipo de datos que no era de propiedad como una [propiedad](../programming-guide/classes-and-structs/properties.md), lo que no está permitido.</span><span class="sxs-lookup"><span data-stu-id="507f8-104">An attempt was made to override a nonproperty data type as a [property](../programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>

```csharp

{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {

      {  
         return 0;  
      }  
   }  
}  
```
