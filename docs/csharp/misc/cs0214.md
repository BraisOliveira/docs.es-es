# <a name="compiler-error-cs0214"></a><span data-ttu-id="95993-102">Error del compilador CS0214</span><span class="sxs-lookup"><span data-stu-id="95993-102">Compiler Error CS0214</span></span>
<span data-ttu-id="95993-103">Los punteros y los búferes de tamaño fijo solo se pueden utilizar en un contexto no seguro</span><span class="sxs-lookup"><span data-stu-id="95993-103">Pointers and fixed size buffers may only be used in an unsafe context</span></span>  
  
 <span data-ttu-id="95993-104">Solo se pueden usar punteros con la palabra clave [unsafe](../language-reference/keywords/unsafe.md) .</span><span class="sxs-lookup"><span data-stu-id="95993-104">Pointers can only be used with the [unsafe](../language-reference/keywords/unsafe.md) keyword.</span></span> <span data-ttu-id="95993-105">Para obtener más información, vea [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md) (Código no seguro y punteros [Guía de programación de C#]).</span><span class="sxs-lookup"><span data-stu-id="95993-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="95993-106">El ejemplo siguiente genera la advertencia CS0214:</span><span class="sxs-lookup"><span data-stu-id="95993-106">The following sample generates CS0214:</span></span>  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
