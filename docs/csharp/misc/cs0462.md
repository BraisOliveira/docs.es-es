# <a name="compiler-error-cs0462"></a><span data-ttu-id="0e691-102">Error del compilador CS0462</span><span class="sxs-lookup"><span data-stu-id="0e691-102">Compiler Error CS0462</span></span>
<span data-ttu-id="0e691-103">Los miembros heredados 'member1' y 'member2' tienen la misma firma en el tipo 'type', por lo que no se pueden reemplazar</span><span class="sxs-lookup"><span data-stu-id="0e691-103">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="0e691-104">Este error se produce con la introducción de genéricos.</span><span class="sxs-lookup"><span data-stu-id="0e691-104">This error arises with the introduction of generics.</span></span> <span data-ttu-id="0e691-105">Normalmente, no se pueden tener dos versiones de un método en una clase con la misma firma.</span><span class="sxs-lookup"><span data-stu-id="0e691-105">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="0e691-106">Pero con los genéricos, es posible especificar un método genérico que podría duplicar a otro si se crea una instancia con un tipo determinado.</span><span class="sxs-lookup"><span data-stu-id="0e691-106">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0e691-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="0e691-107">Example</span></span>  
 <span data-ttu-id="0e691-108">Cuando se crea una instancia de `C<int>` , se crean dos versiones del método `F` con la misma firma, por lo que el reemplazo de la clase `D` no puede decidir a cuál de ellas se debe aplicar el reemplazo.</span><span class="sxs-lookup"><span data-stu-id="0e691-108">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="0e691-109">El ejemplo siguiente genera la advertencia CS0462:</span><span class="sxs-lookup"><span data-stu-id="0e691-109">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>   
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>   
{  
   public override void F(int t) {}   // CS0462  
}  
```
