# <a name="compiler-error-cs0031"></a><span data-ttu-id="d29b7-102">Error del compilador CS0031</span><span class="sxs-lookup"><span data-stu-id="d29b7-102">Compiler Error CS0031</span></span>

<span data-ttu-id="d29b7-103">El valor constante 'valor' no se puede convertir en un 'tipo'.</span><span class="sxs-lookup"><span data-stu-id="d29b7-103">Constant value 'value' cannot be converted to a 'type'.</span></span>

<span data-ttu-id="d29b7-104">Se intentó asignar un valor a una variable cuyo tipo no puede almacenar el valor.</span><span class="sxs-lookup"><span data-stu-id="d29b7-104">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="d29b7-105">Para obtener más información, vea [Tipos](../programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="d29b7-105">For more information, see [Types](../programming-guide/types/index.md).</span></span>

<span data-ttu-id="d29b7-106">El ejemplo siguiente genera la advertencia CS0031 en contextos checked y unchecked:</span><span class="sxs-lookup"><span data-stu-id="d29b7-106">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>

```csharp
// CS0031.cs
namespace CS0031
{
    public class Program
    {
        public static void Main()
        {
            int num = (int)2147483648M; //CS0031
            // Try using a larger numeric type instead.
            // long num = (long)2147483648M; //CS0031

            const decimal d = -10M; // Decimal literal
            unchecked
            {
                const byte b = (byte)d; // CS0031
                // For small values try using a signed byte instead.
                // const sbyte b = (sbyte)d;
            }
        }
    }
}
```
