# <a name="compiler-error-cs0110"></a><span data-ttu-id="7ed6e-102">Error del compilador CS0110</span><span class="sxs-lookup"><span data-stu-id="7ed6e-102">Compiler Error CS0110</span></span>
<span data-ttu-id="7ed6e-103">La evaluación del valor constante para 'const declaration' comprende una definición circular</span><span class="sxs-lookup"><span data-stu-id="7ed6e-103">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="7ed6e-104">La declaración de una variable [const](../../csharp/language-reference/keywords/const.md) (`a`) no puede hacer referencia a otra variable const (`b`) a la que también hace referencia (`a`).</span><span class="sxs-lookup"><span data-stu-id="7ed6e-104">The declaration of a [const](../../csharp/language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="7ed6e-105">El ejemplo siguiente genera la advertencia CS0110:</span><span class="sxs-lookup"><span data-stu-id="7ed6e-105">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
