## <a name="example"></a><span data-ttu-id="fff81-111">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="fff81-111">Example</span></span>  
 <span data-ttu-id="fff81-112">El código siguiente genera el error CS0844:</span><span class="sxs-lookup"><span data-stu-id="fff81-112">The following code generates CS0844:</span></span>  
  
```csharp  
class Test  
    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844  
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="fff81-113">Corrija el error mediante el uso de `this.num` para hacer referencia al campo de clase</span><span class="sxs-lookup"><span data-stu-id="fff81-113">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
