## <a name="example"></a><span data-ttu-id="3138a-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="3138a-107">Example</span></span>  
 <span data-ttu-id="3138a-108">El ejemplo siguiente genera la advertencia CS1657:</span><span class="sxs-lookup"><span data-stu-id="3138a-108">The following sample generates CS1657:</span></span>  
  
```csharp

using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
