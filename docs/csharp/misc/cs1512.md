 <span data-ttu-id="7babb-104">Se usó la palabra clave [base](../../csharp/language-reference/keywords/base.md) fuera de un método, una propiedad o un constructor.</span><span class="sxs-lookup"><span data-stu-id="7babb-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>

```csharp  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
