## <a name="example-1"></a><span data-ttu-id="f8db8-109">Ejemplo 1</span><span class="sxs-lookup"><span data-stu-id="f8db8-109">Example 1</span></span>
 <span data-ttu-id="f8db8-110">El código siguiente genera el error CS0819:</span><span class="sxs-lookup"><span data-stu-id="f8db8-110">The following code generates CS0819:</span></span>
  
```csharp
// cs0819.cs
class Program
{
    public static void Main()
    {
        var a = 3, b = 2; // CS0819

        // First correction option.
        //int a = 3, b = 2;

        // Second correction option.
        //var a = 3;
        //var b = 2;
    }
}
```
