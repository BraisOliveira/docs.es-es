# <a name="compiler-error-cs0155"></a><span data-ttu-id="746f7-102">Error del compilador CS0155</span><span class="sxs-lookup"><span data-stu-id="746f7-102">Compiler Error CS0155</span></span>
<span data-ttu-id="746f7-103">El tipo detectado o producido debe derivarse de System.Exception</span><span class="sxs-lookup"><span data-stu-id="746f7-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="746f7-104">Se intentó pasar un tipo de datos que no se deriva de **System.Exception** en un bloque [catch](../language-reference/keywords/try-catch.md) .</span><span class="sxs-lookup"><span data-stu-id="746f7-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="746f7-105">Solo los tipos de datos que se derivan de **System.Exception** se pueden pasar a un bloque **catch** .</span><span class="sxs-lookup"><span data-stu-id="746f7-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="746f7-106">Para obtener más información, vea [excepciones y control](../programming-guide/exceptions/index.md)de excepciones.</span><span class="sxs-lookup"><span data-stu-id="746f7-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="746f7-107">El ejemplo siguiente genera la advertencia CS0155:</span><span class="sxs-lookup"><span data-stu-id="746f7-107">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
