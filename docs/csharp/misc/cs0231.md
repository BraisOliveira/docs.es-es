# <a name="compiler-error-cs0231"></a><span data-ttu-id="a3150-102">Error del compilador CS0231</span><span class="sxs-lookup"><span data-stu-id="a3150-102">Compiler Error CS0231</span></span>
<span data-ttu-id="a3150-103">Un parámetro params debe ser el último parámetro de una lista de parámetros formales.</span><span class="sxs-lookup"><span data-stu-id="a3150-103">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="a3150-104">El parámetro [params](../../csharp/language-reference/keywords/params.md) admite un número variable de argumentos y debe ser posterior a todos los demás parámetros.</span><span class="sxs-lookup"><span data-stu-id="a3150-104">The [params](../../csharp/language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="a3150-105">Para más información, vea [Métodos](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="a3150-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="a3150-106">El ejemplo siguiente genera la advertencia CS0231:</span><span class="sxs-lookup"><span data-stu-id="a3150-106">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}   
  
   static void Main()   
   {  
   }  
}  
```
