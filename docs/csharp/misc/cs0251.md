# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="da882-102">Advertencia del compilador (nivel 2) CS0251</span><span class="sxs-lookup"><span data-stu-id="da882-102">Compiler Warning (level 2) CS0251</span></span>
<span data-ttu-id="da882-103">Indizando una matriz con un índice negativo (los índices de matriz siempre comienzan por cero)</span><span class="sxs-lookup"><span data-stu-id="da882-103">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="da882-104">No use un número negativo para indexar en una matriz.</span><span class="sxs-lookup"><span data-stu-id="da882-104">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="da882-105">El ejemplo siguiente genera la advertencia CS0251:</span><span class="sxs-lookup"><span data-stu-id="da882-105">The following sample generates CS0251:</span></span>  
  
```csharp  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};     
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
