# <a name="compiler-warning-level-1-cs0626"></a><span data-ttu-id="8caf5-102">Advertencia del compilador (nivel 1) CS0626</span><span class="sxs-lookup"><span data-stu-id="8caf5-102">Compiler Warning (level 1) CS0626</span></span>

<span data-ttu-id="8caf5-103">El método, el operador o el descriptor de acceso 'método' está marcado como externo y no tiene atributos.</span><span class="sxs-lookup"><span data-stu-id="8caf5-103">Method, operator, or accessor 'method' is marked external and has no attributes on it.</span></span> <span data-ttu-id="8caf5-104">Considere la posibilidad de agregar un atributo DllImport para especificar la implementación externa.</span><span class="sxs-lookup"><span data-stu-id="8caf5-104">Consider adding a DllImport attribute to specify the external implementation.</span></span>
  
 <span data-ttu-id="8caf5-105">Un método marcado como `extern` también debe marcarse con un atributo (por ejemplo, el atributo [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) ).</span><span class="sxs-lookup"><span data-stu-id="8caf5-105">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>
  
 <span data-ttu-id="8caf5-106">El atributo especifica dónde se implementa el método.</span><span class="sxs-lookup"><span data-stu-id="8caf5-106">The attribute specifies where the method is implemented.</span></span> <span data-ttu-id="8caf5-107">En tiempo de ejecución, el programa necesitará esta información.</span><span class="sxs-lookup"><span data-stu-id="8caf5-107">At run time, the program will need this information.</span></span>  
  
 <span data-ttu-id="8caf5-108">El ejemplo siguiente genera la advertencia CS0626:</span><span class="sxs-lookup"><span data-stu-id="8caf5-108">The following sample generates CS0626:</span></span>  
  
```csharp
// CS0626.cs  
// compile with: /warnaserror  
using System.Runtime.InteropServices;  
  
public class MyClass  
{  
   static extern public void M(); // CS0626  
   // try the following line  
   // [DllImport("mydll.dll")] static extern public void M();  
  
   public static void Main()  
   {  
   }  
}  
```
