## <a name="example"></a><span data-ttu-id="1b5d4-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="1b5d4-108">Example</span></span>

```csharp  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class   
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
