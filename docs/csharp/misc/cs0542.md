# <a name="compiler-error-cs0542"></a><span data-ttu-id="0e530-102">Error del compilador CS0542</span><span class="sxs-lookup"><span data-stu-id="0e530-102">Compiler Error CS0542</span></span>
<span data-ttu-id="0e530-103">'user-defined type': los nombres de los miembros no pueden coincidir con sus tipos envolventes</span><span class="sxs-lookup"><span data-stu-id="0e530-103">'user-defined type' : member names cannot be the same as their enclosing type</span></span>  
  
 <span data-ttu-id="0e530-104">Los miembros de una clase o una estructura no pueden tener el mismo nombre que la clase o la estructura, a menos que el miembro sea un constructor.</span><span class="sxs-lookup"><span data-stu-id="0e530-104">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>  
  
 <span data-ttu-id="0e530-105">El ejemplo siguiente genera la advertencia CS0542:</span><span class="sxs-lookup"><span data-stu-id="0e530-105">The following sample generates CS0542:</span></span>  
  
```csharp  
// CS0542.cs  
class C  
{  
    public int C;  
}  
```  
  
 <span data-ttu-id="0e530-106">Este error podría producirse si coloca accidentalmente un tipo de valor devuelto en un constructor, que a todos los efectos se convierte en un método normal.</span><span class="sxs-lookup"><span data-stu-id="0e530-106">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="0e530-107">En el ejemplo siguiente se genera el error CS0542 porque `F` es un método, no un constructor, ya que tiene un tipo de valor devuelto:</span><span class="sxs-lookup"><span data-stu-id="0e530-107">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>  
  
```csharp  
// CS0542.cs  
class F  
{  
   // Remove void from F() to resolve the problem.  
   void F()   // CS0542, same name as the class  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="0e530-108">Si la clase se denomina 'Item' y tiene un indizador declarado como `this`, puede que obtenga este error.</span><span class="sxs-lookup"><span data-stu-id="0e530-108">If your class is named 'Item' and has an indexer declared as `this`, you may get this error.</span></span> <span data-ttu-id="0e530-109">Se asigna a un indizador predeterminado el nombre 'Item' en el código emitido, lo que crea el conflicto.</span><span class="sxs-lookup"><span data-stu-id="0e530-109">A default indexer is given the name 'Item' in the emitted code, creating the conflict.</span></span>

   {  
   }
