# <a name="compiler-error-cs0072"></a><span data-ttu-id="34b06-102">Error del compilador CS0072</span><span class="sxs-lookup"><span data-stu-id="34b06-102">Compiler Error CS0072</span></span>
<span data-ttu-id="34b06-103">'evento': no se puede invalidar; 'método' no es un evento.</span><span class="sxs-lookup"><span data-stu-id="34b06-103">'event' : cannot override; 'method' is not an event</span></span>  
  
 <span data-ttu-id="34b06-104">Un [evento](../language-reference/keywords/event.md) solo puede reemplazar a otro evento.</span><span class="sxs-lookup"><span data-stu-id="34b06-104">An [event](../language-reference/keywords/event.md) can only override another event.</span></span> <span data-ttu-id="34b06-105">Para más información, vea [Eventos](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="34b06-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="34b06-106">El ejemplo siguiente genera la advertencia CS0072:</span><span class="sxs-lookup"><span data-stu-id="34b06-106">The following sample generates CS0072:</span></span>  
  
```csharp  
// CS0072.cs  
delegate void MyDelegate();  
  
class Test1  
{  
   public virtual event MyDelegate MyEvent;  
   public virtual void VMeth()  
   {  
   }  
  
   public void FireAway()  
   {  
      if (MyEvent != null)  
         MyEvent();  
   }  
}  
  
class Test2 : Test1  
{  
   public override event MyDelegate VMeth   // CS0072  
   // uncomment the following lines to resolve  
   // public override event MyDelegate MyEvent  
   {  
      add  
      {  
         VMeth += value;  
         // MyEvent += value;  
      }  
      remove  
      {  
         VMeth -= value;  
         // MyEvent -= value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
