<span data-ttu-id="c6824-104">No hay ninguna conversión predefinida entre tipos.</span><span class="sxs-lookup"><span data-stu-id="c6824-104">There is no predefined conversion between types.</span></span> <span data-ttu-id="c6824-105">Puede definir una conversión personalizada entre esos tipos.</span><span class="sxs-lookup"><span data-stu-id="c6824-105">You can define a custom conversion between those types.</span></span> <span data-ttu-id="c6824-106">Para obtener más información, vea [Operadores de conversión definidos por el usuario](../language-reference/operators/user-defined-conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="c6824-106">For more information, see [User-defined conversion operators](../language-reference/operators/user-defined-conversion-operators.md).</span></span>  
  
 <span data-ttu-id="c6824-107">El ejemplo siguiente genera la advertencia CS0030:</span><span class="sxs-lookup"><span data-stu-id="c6824-107">The following sample generates CS0030:</span></span>  
  
```csharp  
// CS0030.cs  
namespace x  
{  
   public class iii  
   {  
      /*  
      public static implicit operator iii(int aa)  
      {  
         return null;  
      }  
  
      public static implicit operator int(iii aa)  
      {  
         return 0;  
      }  
      */  
  
      public static iii operator ++(iii aa)  
      {  
         return (iii)0;   // CS0030  
         // uncomment the conversion routines to resolve CS0030  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
