# <a name="compiler-error-cs0748"></a><span data-ttu-id="559f1-102">Error del compilador CS0748</span><span class="sxs-lookup"><span data-stu-id="559f1-102">Compiler Error CS0748</span></span>

<span data-ttu-id="559f1-103">Uso de parámetros lambda incoherentes; los tipos de parámetro deben ser todos explícitos o implícitos.</span><span class="sxs-lookup"><span data-stu-id="559f1-103">Inconsistent lambda parameter usage; parameter types must be all explicit or all implicit.</span></span>
  
<span data-ttu-id="559f1-104">Si una expresión lambda tiene varios parámetros de entrada, algunos parámetros no pueden usar tipos implícitos mientras que otros usan tipos explícitos.</span><span class="sxs-lookup"><span data-stu-id="559f1-104">If a lambda expression has multiple input parameters, some parameters cannot use implicit typing while others use explicit typing.</span></span>

<span data-ttu-id="559f1-105">Para corregir este error, omita todas las declaraciones de tipos de parámetro o especifique explícitamente el tipo de todos los parámetros.</span><span class="sxs-lookup"><span data-stu-id="559f1-105">To correct this error, either omit all parameter type declarations or explicitly specify the type of all parameters.</span></span>
  
## <a name="example"></a><span data-ttu-id="559f1-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="559f1-106">Example</span></span>
