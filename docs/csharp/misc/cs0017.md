# <a name="compiler-error-cs0017"></a><span data-ttu-id="86af0-102">Error del compilador CS0017</span><span class="sxs-lookup"><span data-stu-id="86af0-102">Compiler Error CS0017</span></span>
<span data-ttu-id="86af0-103">El programa 'nombre de archivo de salida' tiene más de un punto de entrada definido.</span><span class="sxs-lookup"><span data-stu-id="86af0-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="86af0-104">Compile con /main para especificar el tipo que contiene el punto de entrada.</span><span class="sxs-lookup"><span data-stu-id="86af0-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="86af0-105">Un programa puede tener un solo método [Main](../programming-guide/main-and-command-args/index.md) .</span><span class="sxs-lookup"><span data-stu-id="86af0-105">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="86af0-106">Para resolver este error, puede eliminar todos los métodos Main del código, salvo uno, o bien puede usar la opción del compilador [/main](../language-reference/compiler-options/main-compiler-option.md) para especificar qué método Main que se va a usar.</span><span class="sxs-lookup"><span data-stu-id="86af0-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="86af0-107">El ejemplo siguiente genera la advertencia CS0017:</span><span class="sxs-lookup"><span data-stu-id="86af0-107">The following sample generates CS0017:</span></span>  
  
```csharp  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
