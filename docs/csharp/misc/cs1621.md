# <a name="compiler-error-cs1621"></a><span data-ttu-id="7caf9-102">Error del compilador CS1621</span><span class="sxs-lookup"><span data-stu-id="7caf9-102">Compiler Error CS1621</span></span>
<span data-ttu-id="7caf9-103">La instrucción yield no se puede usar dentro de un método anónimo o una expresión lambda</span><span class="sxs-lookup"><span data-stu-id="7caf9-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="7caf9-104">No se puede usar la instrucción [yield](../../csharp/language-reference/keywords/yield.md) en un [método anónimo](../language-reference/operators/delegate-operator.md) o una [expresión lambda](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="7caf9-104">You cannot use the [yield](../../csharp/language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="7caf9-105">El ejemplo siguiente genera la advertencia CS1621:</span><span class="sxs-lookup"><span data-stu-id="7caf9-105">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
