## <a name="example"></a><span data-ttu-id="8e427-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="8e427-107">Example</span></span>

 <span data-ttu-id="8e427-108">En el ejemplo siguiente se genera CS0459 cuando se realiza un intento de tomar la dirección de una variable local de solo lectura en un bucle de `foreach` y en un bloque de instrucciones de `fixed`:</span><span class="sxs-lookup"><span data-stu-id="8e427-108">The following example generates CS0459 when an attempt is made to take the address of a read-only local variable in a `foreach` loop and in a `fixed` statement block:</span></span>

```csharp
// CS0459.cs
// compile with: /unsafe

class Program
{
    public unsafe void M1()
    {
        int[] ints = new int[] { 1, 2, 3 };
        foreach (int i in ints)
        {
            int *j = &i;  // CS0459
        }

        fixed (int *i = &_i)
        {
            int **j = &i;  // CS0459
        }
    }

    private int _i = 0;
}
```
