# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="f9c91-102">Advertencia del compilador (nivel 3) CS0659</span><span class="sxs-lookup"><span data-stu-id="f9c91-102">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="f9c91-103">'clase' invalida Object.Equals(object o), pero no invalida Object.GetHashCode()</span><span class="sxs-lookup"><span data-stu-id="f9c91-103">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="f9c91-104">El compilador detectó una invalidación de la <xref:System.Object.Equals%2A?displayProperty=nameWithType> la invalidación de método, pero no la <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> método.</span><span class="sxs-lookup"><span data-stu-id="f9c91-104">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="f9c91-105">Una invalidación de <xref:System.Object.Equals%2A> implica que también desea reemplazar <xref:System.Object.GetHashCode%2A>.</span><span class="sxs-lookup"><span data-stu-id="f9c91-105">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="f9c91-106">El código siguiente genera la advertencia CS0659:</span><span class="sxs-lookup"><span data-stu-id="f9c91-106">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
