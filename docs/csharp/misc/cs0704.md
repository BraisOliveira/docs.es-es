## <a name="example"></a><span data-ttu-id="f0f24-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="f0f24-106">Example</span></span>

```csharp

{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}

{  
    public static void Main() {}  
}
```
