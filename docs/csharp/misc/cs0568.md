# <a name="compiler-error-cs0568"></a><span data-ttu-id="033da-102">Error del compilador CS0568</span><span class="sxs-lookup"><span data-stu-id="033da-102">Compiler Error CS0568</span></span>
<span data-ttu-id="033da-103">Los structs no pueden contener constructores explícitos sin parámetros</span><span class="sxs-lookup"><span data-stu-id="033da-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="033da-104">Cada [struct](../language-reference/keywords/struct.md) ya tiene un constructor sin parámetros que inicializa el objeto a cero.</span><span class="sxs-lookup"><span data-stu-id="033da-104">Each [struct](../language-reference/keywords/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="033da-105">Por lo tanto, los constructores que se pueden crear para un struct deben tener uno o varios parámetros.</span><span class="sxs-lookup"><span data-stu-id="033da-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="033da-106">El ejemplo siguiente genera la advertencia CS0568:</span><span class="sxs-lookup"><span data-stu-id="033da-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
