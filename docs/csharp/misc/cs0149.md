# <a name="compiler-error-cs0149"></a><span data-ttu-id="e4d03-102">Error del compilador CS0149</span><span class="sxs-lookup"><span data-stu-id="e4d03-102">Compiler Error CS0149</span></span>
<span data-ttu-id="e4d03-103">Se espera un nombre de método</span><span class="sxs-lookup"><span data-stu-id="e4d03-103">Method name expected</span></span>  
  
 <span data-ttu-id="e4d03-104">Al crear un [delegado](../language-reference/keywords/delegate.md), especifique un método.</span><span class="sxs-lookup"><span data-stu-id="e4d03-104">When creating a [delegate](../language-reference/keywords/delegate.md), specify a method.</span></span> <span data-ttu-id="e4d03-105">Para obtener más información, vea [Delegados (Guía de programación de C#)](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="e4d03-105">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="e4d03-106">El ejemplo siguiente genera la advertencia CS0149:</span><span class="sxs-lookup"><span data-stu-id="e4d03-106">The following sample generates CS0149:</span></span>  
  
```csharp  
// CS0149.cs  
using System;  
  
delegate string MyDelegate(int i);  
  
class MyClass  
{  
   // class member-field of the declared delegate type  
   static MyDelegate dt;     
  
   public static void Main()  
   {  
      dt = new MyDelegate(17.45);   // CS0149  
      // try the following line instead  
      // dt = new MyDelegate(Func2);  
      F(dt);  
   }  
  
   public static string Func2(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void F(MyDelegate myFunc)  
   {  
      myFunc(8);  
   }  
}  
```
