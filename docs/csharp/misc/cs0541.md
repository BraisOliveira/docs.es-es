 <span data-ttu-id="a5dd3-104">Una declaración explícita de la [interfaz](../language-reference/keywords/interface.md) solo se puede declarar en una [clase](../language-reference/keywords/class.md) o una [estructura](../language-reference/keywords/struct.md).</span><span class="sxs-lookup"><span data-stu-id="a5dd3-104">An explicit [interface](../language-reference/keywords/interface.md) declaration was found outside a [class](../language-reference/keywords/class.md) or [struct](../language-reference/keywords/struct.md).</span></span>  
  
 <span data-ttu-id="a5dd3-105">El ejemplo siguiente genera la advertencia CS0541:</span><span class="sxs-lookup"><span data-stu-id="a5dd3-105">The following sample generates CS0541:</span></span>  
  
```csharp  
// CS0541.cs

{  
   interface IFace  
   {  
      void F();  
   }

      void IFace.F();   // CS0541  
   }  
}  
```
