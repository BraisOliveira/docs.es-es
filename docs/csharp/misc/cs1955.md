## <a name="example"></a><span data-ttu-id="aa4e8-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="aa4e8-108">Example</span></span>

<span data-ttu-id="aa4e8-109">El código siguiente genera el error CS1955 porque el código intenta invocar un campo y una propiedad mediante el [operador de invocación `()` ](../language-reference/operators/member-access-operators.md#invocation-operator-).</span><span class="sxs-lookup"><span data-stu-id="aa4e8-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation operator `()`](../language-reference/operators/member-access-operators.md#invocation-operator-).</span></span> <span data-ttu-id="aa4e8-110">No puede llamar a un campo o una propiedad.</span><span class="sxs-lookup"><span data-stu-id="aa4e8-110">You cannot call a field or a property.</span></span> <span data-ttu-id="aa4e8-111">Use la [operador de acceso a miembro `.` ](../language-reference/operators/member-access-operators.md#member-access-operator-) para acceder al valor almacena.</span><span class="sxs-lookup"><span data-stu-id="aa4e8-111">Use the [member access operator `.`](../language-reference/operators/member-access-operators.md#member-access-operator-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
