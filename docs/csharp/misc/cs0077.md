# <a name="compiler-error-cs0077"></a><span data-ttu-id="642c1-102">Error del compilador CS0077</span><span class="sxs-lookup"><span data-stu-id="642c1-102">Compiler Error CS0077</span></span>
<span data-ttu-id="642c1-103">El operador as se debe usar con un tipo de referencia o un tipo que acepta valores NULL ('int' es un tipo de valor que no acepta valores NULL)</span><span class="sxs-lookup"><span data-stu-id="642c1-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="642c1-104">Se pasó al operador [as](../language-reference/operators/type-testing-and-cast.md#as-operator) un [tipo de valor](../language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="642c1-104">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="642c1-105">Puesto que `as` puede devolver [null](../language-reference/keywords/null.md), solo se le pueden pasar [tipos de referencia](../language-reference/keywords/reference-types.md) o un tipo que acepta valores NULL.</span><span class="sxs-lookup"><span data-stu-id="642c1-105">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed [reference types](../language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="642c1-106">Para obtener más información sobre los tipos que aceptan valores NULL, vea [tipos que aceptan valores NULL](../programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="642c1-106">For more information about nullable types, see [Nullable Types](../programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="642c1-107">El ejemplo siguiente genera la advertencia CS0077:</span><span class="sxs-lookup"><span data-stu-id="642c1-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
