# <a name="compiler-error-cs0744"></a><span data-ttu-id="ae17f-102">Error del compilador CS0744</span><span class="sxs-lookup"><span data-stu-id="ae17f-102">Compiler Error CS0744</span></span>
<span data-ttu-id="ae17f-103">Se esperaba la palabra clave contextual 'equals'</span><span class="sxs-lookup"><span data-stu-id="ae17f-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="ae17f-104">El patrón de una cláusula `join` es `join`...`in`...`on`...`equals`, como se muestra en este ejemplo:</span><span class="sxs-lookup"><span data-stu-id="ae17f-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

```csharp  
            join y in array2 on x equals y  
            select x;  
```

## <a name="example"></a><span data-ttu-id="ae17f-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="ae17f-107">Example</span></span>  
 <span data-ttu-id="ae17f-108">El código siguiente genera el error CS0744:</span><span class="sxs-lookup"><span data-stu-id="ae17f-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
