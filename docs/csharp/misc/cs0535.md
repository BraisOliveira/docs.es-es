## <a name="example"></a><span data-ttu-id="d2a73-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="d2a73-106">Example</span></span>  
 <span data-ttu-id="d2a73-107">El ejemplo siguiente genera la advertencia CS0535.</span><span class="sxs-lookup"><span data-stu-id="d2a73-107">The following sample generates CS0535.</span></span>  
  
```csharp

   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented

   public void F() {}  
   public static void Main() {}  
}  
```
