 <span data-ttu-id="41db6-106">El ejemplo siguiente genera la advertencia CS0132:</span><span class="sxs-lookup"><span data-stu-id="41db6-106">The following sample generates CS0132:</span></span>  
  
```csharp  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
