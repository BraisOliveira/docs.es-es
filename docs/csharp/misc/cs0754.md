## <a name="example"></a><span data-ttu-id="67890-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="67890-107">Example</span></span>  
 <span data-ttu-id="67890-108">El código siguiente genera el error CS0754:</span><span class="sxs-lookup"><span data-stu-id="67890-108">The following code generates CS0754:</span></span>  
  
```csharp

using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
