# <a name="compiler-error-cs1623"></a><span data-ttu-id="a0d68-102">Error del compilador CS1623</span><span class="sxs-lookup"><span data-stu-id="a0d68-102">Compiler Error CS1623</span></span>
<span data-ttu-id="a0d68-103">Los iteradores no pueden tener parámetros out o ref</span><span class="sxs-lookup"><span data-stu-id="a0d68-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="a0d68-104">Este error se produce si un método iterador toma un `in`, `ref`, o `out` parámetro.</span><span class="sxs-lookup"><span data-stu-id="a0d68-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="a0d68-105">Para evitar este error, quite el `in`, `ref`, o `out` palabra clave de la firma del método.</span><span class="sxs-lookup"><span data-stu-id="a0d68-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a0d68-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="a0d68-106">Example</span></span>  
 <span data-ttu-id="a0d68-107">El ejemplo siguiente genera la advertencia CS1623:</span><span class="sxs-lookup"><span data-stu-id="a0d68-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
