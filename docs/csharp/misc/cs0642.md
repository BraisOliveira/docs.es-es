 <span data-ttu-id="adb11-106">El ejemplo siguiente genera la advertencia CS0642:</span><span class="sxs-lookup"><span data-stu-id="adb11-106">The following sample generates CS0642:</span></span>

{  
   public static void Main()  
   {  
      int i;

      }  
   }
