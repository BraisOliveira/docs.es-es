# <a name="compiler-error-cs0558"></a><span data-ttu-id="5013b-102">Error del compilador CS0558</span><span class="sxs-lookup"><span data-stu-id="5013b-102">Compiler Error CS0558</span></span>

 <span data-ttu-id="5013b-104">Los **modificadores** de acceso **static** y [public](../language-reference/keywords/modifiers.md) deben especificarse ambos en los operadores definidos por el usuario.</span><span class="sxs-lookup"><span data-stu-id="5013b-104">Both the **static** and **public** access [modifiers](../language-reference/keywords/modifiers.md) must be specified on user-defined operators.</span></span>

```csharp  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
