# <a name="compiler-error-cs1055"></a><span data-ttu-id="fa3c4-102">Error del compilador CS1055</span><span class="sxs-lookup"><span data-stu-id="fa3c4-102">Compiler Error CS1055</span></span>
<span data-ttu-id="fa3c4-103">Se esperaba un descriptor de acceso add o remove</span><span class="sxs-lookup"><span data-stu-id="fa3c4-103">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="fa3c4-104">Si su [evento](../language-reference/keywords/event.md) no se ha declarado como campo, debe definir las funciones de descriptor de acceso **add** y **remove** .</span><span class="sxs-lookup"><span data-stu-id="fa3c4-104">If your [event](../language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="fa3c4-105">El ejemplo siguiente genera la advertencia CS1055:</span><span class="sxs-lookup"><span data-stu-id="fa3c4-105">The following sample generates CS1055:</span></span>  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
