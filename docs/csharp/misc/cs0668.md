# <a name="compiler-error-cs0668"></a><span data-ttu-id="579a1-102">Error del compilador CS0668</span><span class="sxs-lookup"><span data-stu-id="579a1-102">Compiler Error CS0668</span></span>
<span data-ttu-id="579a1-103">Dos indexadores tienen nombres distintos; el atributo IndexerName se debe utilizar con el mismo nombre en todos los indexadores de un tipo</span><span class="sxs-lookup"><span data-stu-id="579a1-103">Two indexers have different names; the IndexerName attribute must be used with the same name on every indexer within a type</span></span>  
  
 <span data-ttu-id="579a1-104">Los valores pasados al atributo **IndexerName** deben ser iguales para todos los indexadores de un tipo.</span><span class="sxs-lookup"><span data-stu-id="579a1-104">The values passed to the **IndexerName** attribute must be the same for all indexers in a type.</span></span> <span data-ttu-id="579a1-105">Para más información sobre el atributo **IndexerName** , vea [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute)(Clase IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="579a1-105">For more information on the **IndexerName** attribute, see [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="579a1-106">El ejemplo siguiente genera la advertencia CS0668:</span><span class="sxs-lookup"><span data-stu-id="579a1-106">The following sample generates CS0668:</span></span>  
  
```csharp
// CS0668.cs  
using System;  
using System.Runtime.CompilerServices;  
  
class IndexerClass  
{  
   [IndexerName("IName1")]  
   public int this [int index]   // indexer declaration  
   {  
      get  
      {  
         return index;  
      }  
      set  
      {  
      }  
   }  
  
   [IndexerName("IName2")]  
   public int this [string s]    // CS0668, change IName2 to IName1  
   {  
      get  
      {  
         return int.Parse(s);  
      }  
      set  
      {  
      }  
   }  
  
   void Main()  
   {  
   }  
}  
```
