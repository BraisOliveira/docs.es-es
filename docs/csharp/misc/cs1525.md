# <a name="compiler-error-cs1525"></a><span data-ttu-id="ab67c-102">Error del compilador CS1525</span><span class="sxs-lookup"><span data-stu-id="ab67c-102">Compiler Error CS1525</span></span>
<span data-ttu-id="ab67c-103">Término de la expresión 'character' no válido</span><span class="sxs-lookup"><span data-stu-id="ab67c-103">Invalid expression term 'character'</span></span>  
  
 <span data-ttu-id="ab67c-104">El compilador detectó un carácter no válido en una expresión.</span><span class="sxs-lookup"><span data-stu-id="ab67c-104">The compiler detected an invalid character in an expression.</span></span>  
  
 <span data-ttu-id="ab67c-105">El ejemplo siguiente genera la advertencia CS1525:</span><span class="sxs-lookup"><span data-stu-id="ab67c-105">The following sample generates CS1525:</span></span>  
  
```csharp  
// CS1525.cs  
class x  
{  
   public static void Main()  
   {  
      int i = 0;  
      i = i +   // OK - identifier  
      'c' +     // OK - character  
      (5) +     // OK - parenthesis  
      [ +       // CS1525, operator not a valid expression element  
      throw +   // CS1525, keyword not allowed in expression  
      void;     // CS1525, void not allowed in expression  
   }  
}  
```  
  
 <span data-ttu-id="ab67c-106">Una etiqueta vacía también puede generar el error CS1525, como en el ejemplo siguiente:</span><span class="sxs-lookup"><span data-stu-id="ab67c-106">An empty label can also generate CS1525, as in the following sample:</span></span>  
  
```csharp  
// CS1525b.cs  
using System;  
public class MyClass  
{  
   public static void Main()  
   {  
      goto FoundIt;  
      FoundIt:      // CS1525  
      // Uncomment the following line to resolve:  
      // System.Console.Write("Hello");  
   }  
}  
```
