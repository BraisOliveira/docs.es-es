# <a name="compiler-error-cs0220"></a><span data-ttu-id="646bf-102">Error del compilador CS0220</span><span class="sxs-lookup"><span data-stu-id="646bf-102">Compiler Error CS0220</span></span>
<span data-ttu-id="646bf-103">La operación se desborda en el momento de la compilación en modo checked</span><span class="sxs-lookup"><span data-stu-id="646bf-103">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="646bf-104">Se detectó una operación mediante [checked](../language-reference/keywords/checked.md), que es el valor predeterminado, y tuvo como resultado una pérdida de datos.</span><span class="sxs-lookup"><span data-stu-id="646bf-104">An operation was detected by [checked](../language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="646bf-105">Corrija las entradas de la asignación o use [unchecked](../language-reference/keywords/unchecked.md) para resolver este error.</span><span class="sxs-lookup"><span data-stu-id="646bf-105">Either correct the inputs to the assignment or use [unchecked](../language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="646bf-106">Para obtener más información, vea [Checked y](../language-reference/keywords/checked-and-unchecked.md)Unchecked.</span><span class="sxs-lookup"><span data-stu-id="646bf-106">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="646bf-107">El ejemplo siguiente genera la advertencia CS0220:</span><span class="sxs-lookup"><span data-stu-id="646bf-107">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
