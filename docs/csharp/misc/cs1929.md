## <a name="example"></a><span data-ttu-id="e619f-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="e619f-108">Example</span></span>  
 <span data-ttu-id="e619f-109">El código siguiente genera los errores CS1928 y CS1929:</span><span class="sxs-lookup"><span data-stu-id="e619f-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
