# <a name="compiler-error-cs0218"></a><span data-ttu-id="4ec1c-102">Error del compilador CS0218</span><span class="sxs-lookup"><span data-stu-id="4ec1c-102">Compiler Error CS0218</span></span>
<span data-ttu-id="4ec1c-103">El tipo ('type') debe incluir declaraciones de operador true y operador false</span><span class="sxs-lookup"><span data-stu-id="4ec1c-103">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
<span data-ttu-id="4ec1c-104">Si un tipo definido por el usuario se sobrecarga la [& operador](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) o [ &#124; operador](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), también debe definir [true y false](../language-reference/operators/true-false-operators.md) operadores, con el fin de realizar la evaluación "cortocircuitada" [& & (operador)](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) o [ &#124; &#124; operador](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) definido.</span><span class="sxs-lookup"><span data-stu-id="4ec1c-104">If a user-defined type overloads the [& operator](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) or [&#124; operator](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), it must also define [true and false](../language-reference/operators/true-false-operators.md) operators, in order to make short-circuiting [&& operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) or [&#124;&#124; operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) defined.</span></span>
  
 <span data-ttu-id="4ec1c-105">El ejemplo siguiente genera la advertencia CS0218:</span><span class="sxs-lookup"><span data-stu-id="4ec1c-105">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
