## <a name="example"></a><span data-ttu-id="53d9a-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="53d9a-108">Example</span></span>  
 <span data-ttu-id="53d9a-109">El ejemplo siguiente genera el error CS0835:</span><span class="sxs-lookup"><span data-stu-id="53d9a-109">The following example generates CS0835:</span></span>  
  
```csharp

using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
