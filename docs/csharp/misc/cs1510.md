## <a name="example"></a><span data-ttu-id="8fcae-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="8fcae-106">Example</span></span>

```csharp  
      j++;

   {  
      M (ref 2);   // CS1510, can't pass a number as a ref parameter  
      // try the following to resolve the error  
      // M (ref j);  
   }  
}  
```
