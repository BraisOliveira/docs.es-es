# <a name="compiler-error-cs0557"></a><span data-ttu-id="93141-102">Error del compilador CS0557</span><span class="sxs-lookup"><span data-stu-id="93141-102">Compiler Error CS0557</span></span>
<span data-ttu-id="93141-103">Conversión definida por el usuario duplicada en el tipo 'class'</span><span class="sxs-lookup"><span data-stu-id="93141-103">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="93141-104">No se permiten rutinas de conversión duplicadas en una clase.</span><span class="sxs-lookup"><span data-stu-id="93141-104">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="93141-105">El siguiente ejemplo genera el error CS0557:</span><span class="sxs-lookup"><span data-stu-id="93141-105">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
