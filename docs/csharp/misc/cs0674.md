# <a name="compiler-error-cs0674"></a><span data-ttu-id="35b91-102">Error del compilador CS0674</span><span class="sxs-lookup"><span data-stu-id="35b91-102">Compiler Error CS0674</span></span>
<span data-ttu-id="35b91-103">No use 'System.ParamArrayAttribute'.</span><span class="sxs-lookup"><span data-stu-id="35b91-103">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="35b91-104">Use la palabra clave 'params' en su lugar.</span><span class="sxs-lookup"><span data-stu-id="35b91-104">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="35b91-105">El compilador de C# no permite el uso de <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) en su lugar.</span><span class="sxs-lookup"><span data-stu-id="35b91-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="35b91-106">El ejemplo siguiente genera la advertencia CS0674:</span><span class="sxs-lookup"><span data-stu-id="35b91-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
