## <a name="example"></a><span data-ttu-id="d0477-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="d0477-108">Example</span></span>  
 <span data-ttu-id="d0477-109">El ejemplo siguiente genera la advertencia CS0559.</span><span class="sxs-lookup"><span data-stu-id="d0477-109">The following sample generates CS0559.</span></span>  
  
```csharp  
// CS0559_b.cs  
// compile with: /target:library  
public struct S  
{  
   public static S operator ++(S? s) { return new S(); }   // CS0559

{

   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
}  
```
