## <a name="example"></a><span data-ttu-id="67bfa-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="67bfa-108">Example</span></span>

<span data-ttu-id="67bfa-109">En el ejemplo siguiente se genera CS4009:</span><span class="sxs-lookup"><span data-stu-id="67bfa-109">The following example produces CS4009:</span></span>

```csharp
using System;
using System.Threading.Tasks;

public class Example
{
    public static async void Main()
    {
        Console.WriteLine("About to wait two seconds");
        await WaitTwoSeconds();
        Console.WriteLine("About to exit the program");
    }

    private static async Task WaitTwoSeconds()
    {
        await Task.Delay(2000);
        Console.WriteLine("Returning from an asynchronous method");
    } 
}
```
