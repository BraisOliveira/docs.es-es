# <a name="compiler-error-cs0538"></a><span data-ttu-id="437fb-102">Error del compilador CS0538</span><span class="sxs-lookup"><span data-stu-id="437fb-102">Compiler Error CS0538</span></span>
<span data-ttu-id="437fb-103">'nombre' en la declaración explícita de la interfaz no es una interfaz.</span><span class="sxs-lookup"><span data-stu-id="437fb-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="437fb-104">Se intentó declarar explícitamente una [interface](../language-reference/keywords/interface.md), pero no se especificó una interfaz.</span><span class="sxs-lookup"><span data-stu-id="437fb-104">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="437fb-105">El ejemplo siguiente genera la advertencia CS0538:</span><span class="sxs-lookup"><span data-stu-id="437fb-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
