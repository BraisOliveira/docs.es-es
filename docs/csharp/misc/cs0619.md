# <a name="compiler-error-cs0619"></a><span data-ttu-id="134d4-102">Error del compilador CS0619</span><span class="sxs-lookup"><span data-stu-id="134d4-102">Compiler Error CS0619</span></span>
<span data-ttu-id="134d4-103">'member' está obsoleto: 'text'</span><span class="sxs-lookup"><span data-stu-id="134d4-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="134d4-104">Un miembro de clase se marcó con el [obsoleto] ((.. /Programming-Guide/Concepts/Attributes/Common-Attributes.MD # obsoleto), de modo que se emitirá un error cuando se haga referencia al miembro de clase.</span><span class="sxs-lookup"><span data-stu-id="134d4-104">A class member was marked with the [Obsolete]((../programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="134d4-105">El ejemplo siguiente genera la advertencia CS0619:</span><span class="sxs-lookup"><span data-stu-id="134d4-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
