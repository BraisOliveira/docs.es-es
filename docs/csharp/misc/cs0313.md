## <a name="to-correct-this-error"></a><span data-ttu-id="0551d-108">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="0551d-108">To correct this error</span></span>

    ```csharp

    ```  
  
## <a name="example"></a><span data-ttu-id="0551d-111">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="0551d-111">Example</span></span>

<span data-ttu-id="0551d-112">El código siguiente genera el error CS0313:</span><span class="sxs-lookup"><span data-stu-id="0551d-112">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
