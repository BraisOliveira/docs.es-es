# <a name="compiler-error-cs1613"></a><span data-ttu-id="99605-102">Error del compilador CS1613</span><span class="sxs-lookup"><span data-stu-id="99605-102">Compiler Error CS1613</span></span>
<span data-ttu-id="99605-103">No se encuentra la clase contenedora de coclases administrada de la interfaz 'interface' (¿falta alguna referencia de ensamblado?)</span><span class="sxs-lookup"><span data-stu-id="99605-103">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="99605-104">Se intentó crear una instancia de un objeto COM de una interfaz.</span><span class="sxs-lookup"><span data-stu-id="99605-104">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="99605-105">La interfaz tiene los atributos **ComImport** y `CoClass` , pero el compilador no puede encontrar el tipo dado para el atributo `CoClass` .</span><span class="sxs-lookup"><span data-stu-id="99605-105">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="99605-106">Para solucionar este error, pruebe una de las propuestas siguientes:</span><span class="sxs-lookup"><span data-stu-id="99605-106">To resolve this error, you can try one of the following:</span></span>  
  
- <span data-ttu-id="99605-107">Agregue una referencia al ensamblado que tenga la coclase (la mayoría de las veces, la interfaz y la coclase deben estar en el mismo ensamblado).</span><span class="sxs-lookup"><span data-stu-id="99605-107">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="99605-108">Vea [: referencia](../language-reference/compiler-options/reference-compiler-option.md) o [Agregar referencia (cuadro de diálogo)](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) para obtener información.</span><span class="sxs-lookup"><span data-stu-id="99605-108">See [-reference](../language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) for information.</span></span>  
  
- <span data-ttu-id="99605-109">Corrija el atributo `CoClass` en la interfaz.</span><span class="sxs-lookup"><span data-stu-id="99605-109">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="99605-110">El siguiente ejemplo muestra el uso correcto de **CoClassAttribute**:</span><span class="sxs-lookup"><span data-stu-id="99605-110">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
