<span data-ttu-id="4eb74-103">'miembro' en la declaración explícita de la interfaz no es un miembro de interfaz.</span><span class="sxs-lookup"><span data-stu-id="4eb74-103">'member' in explicit interface declaration is not a member of interface</span></span>  
  
 <span data-ttu-id="4eb74-104">Se intentó declarar explícitamente un miembro de [interfaz](../language-reference/keywords/interface.md) que no existe.</span><span class="sxs-lookup"><span data-stu-id="4eb74-104">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md) member that does not exist.</span></span> <span data-ttu-id="4eb74-105">Debe eliminar la declaración o cambiarla para que haga referencia a un miembro de interfaz válido.</span><span class="sxs-lookup"><span data-stu-id="4eb74-105">You should either delete the declaration or change it so that it refers to a valid interface member.</span></span>  
  
 <span data-ttu-id="4eb74-106">El ejemplo siguiente genera la advertencia CS0539:</span><span class="sxs-lookup"><span data-stu-id="4eb74-106">The following sample generates CS0539:</span></span>  
  
```csharp  
// CS0539.cs

   interface I  
   {  
      void m();  
   }  
  
   public class clx : I  
   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
