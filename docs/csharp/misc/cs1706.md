## <a name="example"></a><span data-ttu-id="3833c-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="3833c-107">Example</span></span>  
 <span data-ttu-id="3833c-108">El ejemplo siguiente genera el error CS1706.</span><span class="sxs-lookup"><span data-stu-id="3833c-108">The following example generates CS1706.</span></span>

```csharp  
{  
    public MyAttribute(MyDelegate d) { }  
}

[MyAttribute(delegate{/* anonymous Method in Attribute declaration */})]  // CS1706

{  
}  
```
