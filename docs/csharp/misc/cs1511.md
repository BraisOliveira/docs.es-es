# <a name="compiler-error-cs1511"></a><span data-ttu-id="50ac5-102">Error del compilador CS1511</span><span class="sxs-lookup"><span data-stu-id="50ac5-102">Compiler Error CS1511</span></span>
<span data-ttu-id="50ac5-103">La palabra clave 'base' no está disponible en ningún método estático</span><span class="sxs-lookup"><span data-stu-id="50ac5-103">Keyword 'base' is not available in a static method</span></span>  
  
 <span data-ttu-id="50ac5-104">La palabra clave [base](../../csharp/language-reference/keywords/base.md) se usaba en un método [static](../../csharp/language-reference/keywords/static.md) .</span><span class="sxs-lookup"><span data-stu-id="50ac5-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used in a [static](../../csharp/language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="50ac5-105">`base` solo se puede llamar en un constructor de instancia, un método de instancia o un descriptor de acceso de instancia.</span><span class="sxs-lookup"><span data-stu-id="50ac5-105">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="50ac5-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="50ac5-106">Example</span></span>  
 <span data-ttu-id="50ac5-107">El ejemplo siguiente genera la advertencia CS1511.</span><span class="sxs-lookup"><span data-stu-id="50ac5-107">The following sample generates CS1511.</span></span>  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
