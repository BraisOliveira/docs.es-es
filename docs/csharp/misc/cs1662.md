# <a name="compiler-error-cs1662"></a><span data-ttu-id="1a845-102">Error del compilador CS1662</span><span class="sxs-lookup"><span data-stu-id="1a845-102">Compiler Error CS1662</span></span>

<span data-ttu-id="1a845-103">No se puede convertir el bloque de método anónimo al tipo delegado 'tipo delegado' porque algunos de los tipos de valor devuelto del bloque no se pueden convertir implícitamente al tipo de valor devuelto del delegado.</span><span class="sxs-lookup"><span data-stu-id="1a845-103">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>

<span data-ttu-id="1a845-104">Este error se produce si la instrucción return del bloque de método anónimo tenía un tipo que no se puede convertir implícitamente al tipo de valor devuelto del delegado.</span><span class="sxs-lookup"><span data-stu-id="1a845-104">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>

<span data-ttu-id="1a845-105">El ejemplo siguiente genera la advertencia CS1662:</span><span class="sxs-lookup"><span data-stu-id="1a845-105">The following sample generates CS1662:</span></span>

```csharp
// CS1662.cs

delegate int MyDelegate(int i);

class C
{

  public static void Main()
  {
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662
     // Try this instead:
     // MyDelegate d = delegate(int i) { return (int)1.0; };
  }
}
```
