 <span data-ttu-id="66410-104">Una palabra clave **add** o **remove** en una definición [evento](../../csharp/language-reference/keywords/event.md) debe tener un cuerpo.</span><span class="sxs-lookup"><span data-stu-id="66410-104">An **add** or **remove** keyword in an [event](../../csharp/language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="66410-105">Para más información, vea [Eventos](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="66410-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="66410-106">El ejemplo siguiente genera la advertencia CS0073:</span><span class="sxs-lookup"><span data-stu-id="66410-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
