<span data-ttu-id="6ca98-103">Un descriptor de acceso add o remove debe tener un cuerpo</span><span class="sxs-lookup"><span data-stu-id="6ca98-103">An add or remove accessor must have a body</span></span>  
  
 <span data-ttu-id="6ca98-104">Una palabra clave **add** o **remove** en una definición [evento](../language-reference/keywords/event.md) debe tener un cuerpo.</span><span class="sxs-lookup"><span data-stu-id="6ca98-104">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="6ca98-105">Para más información, vea [Eventos](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="6ca98-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="6ca98-106">El ejemplo siguiente genera la advertencia CS0073:</span><span class="sxs-lookup"><span data-stu-id="6ca98-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
