# <a name="compiler-error-cs0070"></a><span data-ttu-id="027c0-102">Error del compilador CS0070</span><span class="sxs-lookup"><span data-stu-id="027c0-102">Compiler Error CS0070</span></span>
<span data-ttu-id="027c0-103">El evento 'event' solo puede aparecer a la izquierda de += o -= (excepto cuando se usa desde el tipo 'type')</span><span class="sxs-lookup"><span data-stu-id="027c0-103">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="027c0-104">Fuera de la clase en la que está definido, un [evento](../language-reference/keywords/event.md) solo puede agregar o restar referencias.</span><span class="sxs-lookup"><span data-stu-id="027c0-104">Outside of the class it is defined in, an [event](../language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="027c0-105">Para más información, vea [Eventos](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="027c0-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="027c0-106">El ejemplo siguiente genera la advertencia CS0070:</span><span class="sxs-lookup"><span data-stu-id="027c0-106">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
