 <span data-ttu-id="b40c7-104">Una construcción [public](../language-reference/keywords/public.md) debe devolver un objeto accesible públicamente.</span><span class="sxs-lookup"><span data-stu-id="b40c7-104">A [public](../language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>

 <span data-ttu-id="b40c7-107">El ejemplo siguiente genera la advertencia CS0061.</span><span class="sxs-lookup"><span data-stu-id="b40c7-107">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
