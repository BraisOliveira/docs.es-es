# <a name="compiler-error-cs0156"></a><span data-ttu-id="36029-102">Error del compilador CS0156</span><span class="sxs-lookup"><span data-stu-id="36029-102">Compiler Error CS0156</span></span>
<span data-ttu-id="36029-103">No se permite una instrucción throw sin argumentos en una cláusula finally anidada en la cláusula catch más cercana</span><span class="sxs-lookup"><span data-stu-id="36029-103">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="36029-104">Una instrucción [throw](../../csharp/language-reference/keywords/throw.md) sin parámetros solo puede aparecer en una cláusula **catch** que no tome parámetros.</span><span class="sxs-lookup"><span data-stu-id="36029-104">A [throw](../../csharp/language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="36029-105">Para obtener más información, consulte [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) y [excepciones y control de excepciones](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="36029-105">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="36029-106">El ejemplo siguiente genera la advertencia CS0156:</span><span class="sxs-lookup"><span data-stu-id="36029-106">The following sample generates CS0156:</span></span>  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
