## <a name="example"></a><span data-ttu-id="a03b8-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="a03b8-106">Example</span></span>

 <span data-ttu-id="a03b8-107">El ejemplo siguiente genera la advertencia CS1648:</span><span class="sxs-lookup"><span data-stu-id="a03b8-107">The following sample generates CS1648:</span></span>

```csharp
// CS1648.cs
public struct Inner
{
    public int i;
}

class Outer
{
    public readonly Inner inner = new Inner();
}

class D
{
    static void Main()
    {
        var outer = new Outer();
        outer.inner.i = 1;  // CS1648
    }
}
```
