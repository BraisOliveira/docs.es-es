## <a name="example"></a><span data-ttu-id="9aa52-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="9aa52-107">Example</span></span>

```csharp  
[assembly: CLSCompliant(true)]

    [CLSCompliant(false)]  // CS3019

    }

    }  
}  
```
