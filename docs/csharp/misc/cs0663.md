# <a name="compiler-error-cs0663"></a><span data-ttu-id="a5ede-102">Error del compilador CS0663</span><span class="sxs-lookup"><span data-stu-id="a5ede-102">Compiler Error CS0663</span></span>
<span data-ttu-id="a5ede-103">No se pueden definir métodos sobrecargados que difieran solo en ref y out.</span><span class="sxs-lookup"><span data-stu-id="a5ede-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="a5ede-104">Los métodos que se diferencian solo en el uso de [en](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) y [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) no se permiten en un parámetro.</span><span class="sxs-lookup"><span data-stu-id="a5ede-104">Methods that differ only on their use of [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="a5ede-105">El ejemplo siguiente genera la advertencia CS0663:</span><span class="sxs-lookup"><span data-stu-id="a5ede-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
