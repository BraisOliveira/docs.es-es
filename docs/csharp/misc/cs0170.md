<span data-ttu-id="f94ff-103">Uso del campo 'field' posiblemente sin asignar</span><span class="sxs-lookup"><span data-stu-id="f94ff-103">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="f94ff-104">Se usó un campo de una estructura sin inicializarlo primero.</span><span class="sxs-lookup"><span data-stu-id="f94ff-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="f94ff-105">Para solucionar este problema, determine primero de qué campo se canceló la inicialización e inicialícelo antes de intentar acceder a este.</span><span class="sxs-lookup"><span data-stu-id="f94ff-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="f94ff-106">Para obtener más información sobre la inicialización de Structs, vea [Structs](../programming-guide/classes-and-structs/structs.md) and [using Structs](../programming-guide/classes-and-structs/using-structs.md).</span><span class="sxs-lookup"><span data-stu-id="f94ff-106">For more information about initializing structs, see [Structs](../programming-guide/classes-and-structs/structs.md) and [Using Structs](../programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
 <span data-ttu-id="f94ff-107">El ejemplo siguiente genera la advertencia CS0170:</span><span class="sxs-lookup"><span data-stu-id="f94ff-107">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
