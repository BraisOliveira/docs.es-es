# <a name="compiler-error-cs1558"></a><span data-ttu-id="d28df-102">Error del compilador CS1558</span><span class="sxs-lookup"><span data-stu-id="d28df-102">Compiler Error CS1558</span></span>
<span data-ttu-id="d28df-103">'clase' no tiene ningún método Main estático adecuado</span><span class="sxs-lookup"><span data-stu-id="d28df-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="d28df-104">La opción del compilador [-Main](../language-reference/compiler-options/main-compiler-option.md) especificó una clase en la que buscar un método **Main** .</span><span class="sxs-lookup"><span data-stu-id="d28df-104">The [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="d28df-105">Sin embargo, el método [Main](../programming-guide/main-and-command-args/index.md) no se definió correctamente.</span><span class="sxs-lookup"><span data-stu-id="d28df-105">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="d28df-106">En el siguiente ejemplo se genera el error CS1558 debido a un tipo de valor devuelto no válido.</span><span class="sxs-lookup"><span data-stu-id="d28df-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
