# <a name="compiler-error-cs1922"></a><span data-ttu-id="c9a20-102">Error del compilador CS1922</span><span class="sxs-lookup"><span data-stu-id="c9a20-102">Compiler Error CS1922</span></span>

<span data-ttu-id="c9a20-103">El inicializador de colección requiere que su tipo 'type' implemente System.Collections.IEnumerable.</span><span class="sxs-lookup"><span data-stu-id="c9a20-103">Collection initializer requires its type 'type' to implement System.Collections.IEnumerable.</span></span>

<span data-ttu-id="c9a20-104">Para usar un inicializador de colección con un tipo, el tipo debe implementar `IEnumerable`.</span><span class="sxs-lookup"><span data-stu-id="c9a20-104">In order to use a collection initializer with a type, the type must implement `IEnumerable`.</span></span> <span data-ttu-id="c9a20-105">Este error puede producirse si se usa accidentalmente sintaxis de inicializador de colección cuando se pretendía usar un inicializador de objeto.</span><span class="sxs-lookup"><span data-stu-id="c9a20-105">This error can occur if you accidentally use collection initializer syntax when you meant to use an object initializer.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="c9a20-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="c9a20-106">To correct this error</span></span>

- <span data-ttu-id="c9a20-107">Si el tipo no representa ninguna colección, use la sintaxis de inicializador de objeto en lugar de la sintaxis del inicializador de colección.</span><span class="sxs-lookup"><span data-stu-id="c9a20-107">If the type does not represent a collection, use object initializer syntax instead of collection initializer syntax.</span></span>

- <span data-ttu-id="c9a20-108">Si el tipo representa una colección, modifíquelo para implementar `IEnumerable` antes de usar inicializadores de colección para inicializar objetos de ese tipo.</span><span class="sxs-lookup"><span data-stu-id="c9a20-108">If the type does represent a collection, modify it to implement `IEnumerable` before you can use collection initializers to initialize objects of that type.</span></span>

- <span data-ttu-id="c9a20-109">Si el tipo representa una colección y no tiene acceso al código fuente, simplemente se inicializan sus elementos mediante el uso de sus constructores de clases u otros métodos de inicialización.</span><span class="sxs-lookup"><span data-stu-id="c9a20-109">If the type represents a collection and you do not have access to the source code, just initialize its elements by using their class constructors or other initialization methods.</span></span>

## <a name="example"></a><span data-ttu-id="c9a20-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="c9a20-110">Example</span></span>

<span data-ttu-id="c9a20-111">El código siguiente produce el error CS1922:</span><span class="sxs-lookup"><span data-stu-id="c9a20-111">The following code produces CS1922:</span></span>

```csharp
// cs1922.cs
public class Test
{
    public static void Main()
    {
        // Collection initializer.
        var tc = new TestClass  {1,"hello"} ; // CS1922

        // Object initializer.
        var tc2 = new TestClass { memberA = 1, memberB = "hello" }; // OK
    }
}

public class TestClass
{
    public int memberA { get; set; }
    public string memberB { get; set; }
}
```
