# <a name="compiler-error-cs0243"></a><span data-ttu-id="96d3f-102">Error del compilador CS0243</span><span class="sxs-lookup"><span data-stu-id="96d3f-102">Compiler Error CS0243</span></span>

<span data-ttu-id="96d3f-103">El atributo Conditional no es válido en 'método' porque es un método de invalidación.</span><span class="sxs-lookup"><span data-stu-id="96d3f-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="96d3f-104">El atributo [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) no se permite en un método marcado con la palabra clave [override](../../csharp/language-reference/keywords/override.md) .</span><span class="sxs-lookup"><span data-stu-id="96d3f-104">The [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="96d3f-105">Para obtener más información, consulte [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="96d3f-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="96d3f-106">El compilador nunca se enlaza a métodos de invalidación; solo se enlaza al método base y Common Language Runtime llama a la invalidación, según corresponda.</span><span class="sxs-lookup"><span data-stu-id="96d3f-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="96d3f-107">El ejemplo siguiente genera la advertencia CS0243:</span><span class="sxs-lookup"><span data-stu-id="96d3f-107">The following sample generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
