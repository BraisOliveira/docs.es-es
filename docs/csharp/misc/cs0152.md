<span data-ttu-id="8fb62-103">La etiqueta 'label' ya se da en esta instrucción switch</span><span class="sxs-lookup"><span data-stu-id="8fb62-103">The label 'label' already occurs in this switch statement</span></span>  
  
 <span data-ttu-id="8fb62-104">Se repitió una etiqueta en una instrucción [switch](../../csharp/language-reference/keywords/switch.md) .</span><span class="sxs-lookup"><span data-stu-id="8fb62-104">A label was repeated in a [switch](../../csharp/language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="8fb62-105">Para obtener más información, consulta [switch](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="8fb62-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="8fb62-106">El ejemplo siguiente genera la advertencia CS0152:</span><span class="sxs-lookup"><span data-stu-id="8fb62-106">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
