 <span data-ttu-id="4af7a-104">No se puede completar una asignación. Use un sufijo para corregir la instrucción.</span><span class="sxs-lookup"><span data-stu-id="4af7a-104">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="4af7a-105">La documentación de cada tipo identifica el sufijo que le corresponde.</span><span class="sxs-lookup"><span data-stu-id="4af7a-105">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="4af7a-106">Para obtener más información sobre las conversiones, vea [conversiones de tipos](../../csharp/programming-guide/types/casting-and-type-conversions.md).</span><span class="sxs-lookup"><span data-stu-id="4af7a-106">For more information on conversions, see [Casting and Type Conversions](../../csharp/programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="4af7a-107">El ejemplo siguiente genera la advertencia CS0664:</span><span class="sxs-lookup"><span data-stu-id="4af7a-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
