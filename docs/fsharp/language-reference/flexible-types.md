# <a name="flexible-types"></a><span data-ttu-id="3855c-103">Tipos flexibles</span><span class="sxs-lookup"><span data-stu-id="3855c-103">Flexible Types</span></span>

<span data-ttu-id="3855c-104">Un *anotación de tipo flexible* indica que un parámetro, una variable o un valor tiene un tipo que es compatible con un tipo especificado, donde compatibilidad viene determinada por la posición en una jerarquía orientada a objetos de clases o interfaces.</span><span class="sxs-lookup"><span data-stu-id="3855c-104">A *flexible type annotation* indicates that a parameter, variable, or value has a type that is compatible with a specified type, where compatibility is determined by position in an object-oriented hierarchy of classes or interfaces.</span></span> <span data-ttu-id="3855c-105">Tipos flexibles son útiles en concreto cuando no se produce la conversión automática a tipos más arriba en la jerarquía de tipos, pero desea habilitar la funcionalidad para trabajar con cualquier tipo de la jerarquía o cualquier tipo que implementa una interfaz.</span><span class="sxs-lookup"><span data-stu-id="3855c-105">Flexible types are useful specifically when the automatic conversion to types higher in the type hierarchy does not occur but you still want to enable your functionality to work with any type in the hierarchy or any type that implements an interface.</span></span>

```fsharp
#type
```
