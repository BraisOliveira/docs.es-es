# <a name="type-providers"></a><span data-ttu-id="9a09b-103">Proveedores de tipos</span><span class="sxs-lookup"><span data-stu-id="9a09b-103">Type Providers</span></span>

<span data-ttu-id="9a09b-104">Un proveedor de tipos de F# es un componente que proporciona tipos, propiedades y métodos para usar en el programa.</span><span class="sxs-lookup"><span data-stu-id="9a09b-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="9a09b-105">Los proveedores de tipos generan lo que se conoce como **proporciona tipos**, que son generados por el F# compilador y se basan en un origen de datos externo.</span><span class="sxs-lookup"><span data-stu-id="9a09b-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="9a09b-106">Por ejemplo, un F# proveedor de tipos para SQL puede generar tipos que representan las tablas y columnas en una base de datos relacional.</span><span class="sxs-lookup"><span data-stu-id="9a09b-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="9a09b-107">De hecho, esto es lo que el [SQLProvider](https://fsprojects.github.io/SQLProvider/) hace el proveedor de tipo.</span><span class="sxs-lookup"><span data-stu-id="9a09b-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="9a09b-108">Proporciona que tipos dependen de los parámetros de entrada a un proveedor de tipos.</span><span class="sxs-lookup"><span data-stu-id="9a09b-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="9a09b-109">Dicha entrada puede ser un origen de datos de ejemplo (por ejemplo, un archivo de esquema JSON), una dirección URL que apunta directamente a un servicio externo o una cadena de conexión a un origen de datos.</span><span class="sxs-lookup"><span data-stu-id="9a09b-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="9a09b-110">Un proveedor de tipos también puede asegurarse de que solo se expanden los grupos de tipos a petición; es decir, que se expandan si realmente se hace referencia a los tipos en los programas.</span><span class="sxs-lookup"><span data-stu-id="9a09b-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="9a09b-111">Esto permite la integración directa a petición de espacios de información a gran escala, tales como mercados de datos en línea de forma fuertemente tipada.</span><span class="sxs-lookup"><span data-stu-id="9a09b-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="9a09b-112">Proveedores de tipos generativa y borrados</span><span class="sxs-lookup"><span data-stu-id="9a09b-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="9a09b-113">Los proveedores de tipos se dividen en dos formas: Generativa y borrados.</span><span class="sxs-lookup"><span data-stu-id="9a09b-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="9a09b-114">Los proveedores de tipos generativa generar tipos que se pueden escribir como tipos de .NET en el ensamblado en la que se generan.</span><span class="sxs-lookup"><span data-stu-id="9a09b-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="9a09b-115">Esto les permite consumir desde el código de otros ensamblados.</span><span class="sxs-lookup"><span data-stu-id="9a09b-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="9a09b-116">Esto significa que la representación del origen de datos con tipo generalmente debe ser uno que sea posible para representar con tipos de. NET.</span><span class="sxs-lookup"><span data-stu-id="9a09b-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="9a09b-117">Borrado de los proveedores de tipos producen tipos que sólo pueden utilizarse en el ensamblado o proyecto que se generan a partir.</span><span class="sxs-lookup"><span data-stu-id="9a09b-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="9a09b-118">Los tipos son efímeros; es decir, que no se escriben en un ensamblado y no pueden usarse por código de otros ensamblados.</span><span class="sxs-lookup"><span data-stu-id="9a09b-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="9a09b-119">Pueden contener *retrasa* miembros, lo que le permite utilizar proporcionado tipos de un espacio de información potencialmente infinito.</span><span class="sxs-lookup"><span data-stu-id="9a09b-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="9a09b-120">Son útiles para el uso de un pequeño subconjunto de un origen de datos de gran tamaño e interconectadas.</span><span class="sxs-lookup"><span data-stu-id="9a09b-120">They are useful for using a small subset of a large and interconnected data source.</span></span>

- <span data-ttu-id="9a09b-125">[FSharp.Data.SqlClient](https://fsprojects.github.io/FSharp.Data.SqlClient/) ha comprobado un conjunto de proveedores de tipo de tiempo de compilación de inserción de T-SQL en F#.</span><span class="sxs-lookup"><span data-stu-id="9a09b-125">[FSharp.Data.SqlClient](https://fsprojects.github.io/FSharp.Data.SqlClient/) has a set of type providers for compile-time checked embedding of T-SQL in F#.</span></span>
