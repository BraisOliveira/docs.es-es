 <span data-ttu-id="4a8a4-134">Cuando use el modo <xref:System.Runtime.GCLatencyMode.LowLatency>, tenga en cuenta las siguientes directrices:</span><span class="sxs-lookup"><span data-stu-id="4a8a4-134">When you use <xref:System.Runtime.GCLatencyMode.LowLatency> mode, consider the following guidelines:</span></span>

- <span data-ttu-id="4a8a4-138">Mientras esté en modo de latencia baja, minimice el número de asignaciones que realiza, en particular las asignaciones en el montón de objetos grandes y de objetos anclados.</span><span class="sxs-lookup"><span data-stu-id="4a8a4-138">While in the low latency mode, minimize the number of allocations you make, in particular allocations onto the Large Object Heap and pinned objects.</span></span>
