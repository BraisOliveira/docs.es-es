## <a name="a-simple-analogy"></a><span data-ttu-id="26043-141">Una analogía simple</span><span class="sxs-lookup"><span data-stu-id="26043-141">A simple analogy</span></span>

<span data-ttu-id="26043-143">Vamos a remontarnos a la década de 1950 por un momento.</span><span class="sxs-lookup"><span data-stu-id="26043-143">Let's go back in time to the 1950s for a moment.</span></span> <span data-ttu-id="26043-144">No había ningún procesadores de textos y el fotocopiadoras se usaron en todas partes (bueno, más).</span><span class="sxs-lookup"><span data-stu-id="26043-144">There were no word processors, and the photocopiers were used everywhere (well, kind of).</span></span>

<span data-ttu-id="26043-145">Imagine que es responsable de enviar lotes de cartas, según proceda, para enviarlas por correo a los clientes en papel y sobres reales, que se entregarán físicamente en la dirección postal de cada cliente (entonces no existía el correo electrónico).</span><span class="sxs-lookup"><span data-stu-id="26043-145">Imagine you're responsible for quickly issuing batches of letters as required, to mail them to customers, using real paper and envelopes, to be delivered physically to each customer's address (there was no email back then).</span></span>

<span data-ttu-id="26043-146">En algún momento, se da cuenta de que las letras no son más que una composición de un conjunto grande de párrafos, que se eligen y ordenan según proceda, según el propósito de la carga, por lo que diseña un sistema para emitir cartas rápidamente, esperando conseguir una increíble mejora.</span><span class="sxs-lookup"><span data-stu-id="26043-146">At some point, you realize the letters are just a composition of a large set of paragraphs, which are picked and arranged as needed, according to the purpose of the letter, so you devise a system to issue letters quickly, expecting to get a hefty raise.</span></span>

<span data-ttu-id="26043-147">El sistema es simple:</span><span class="sxs-lookup"><span data-stu-id="26043-147">The system is simple:</span></span>
