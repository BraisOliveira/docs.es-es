## <a name="how-to-specify-target-frameworks"></a><span data-ttu-id="01c01-195">Cómo especificar plataformas de destino</span><span class="sxs-lookup"><span data-stu-id="01c01-195">How to specify target frameworks</span></span>

<span data-ttu-id="01c01-196">Las plataformas de destino se especifican en el archivo del proyecto.</span><span class="sxs-lookup"><span data-stu-id="01c01-196">Target frameworks are specified in your project file.</span></span> <span data-ttu-id="01c01-197">Cuando especifique una única plataforma de destino, use el elemento **TargetFramework**.</span><span class="sxs-lookup"><span data-stu-id="01c01-197">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="01c01-198">En el siguiente archivo de proyecto de aplicación de consola se muestra cómo elegir como destino .NET Core 2.2:</span><span class="sxs-lookup"><span data-stu-id="01c01-198">The following console app project file demonstrates how to target .NET Core 2.2:</span></span>

```xml

  <PropertyGroup>
    <OutputType>Exe</OutputType>
    <TargetFramework>netcoreapp2.2</TargetFramework>
  </PropertyGroup>

</Project>
```

<span data-ttu-id="01c01-199">Al especificar varias plataformas de destino, puede hacer referencia de forma condicional a ensamblados para cada plataforma de destino.</span><span class="sxs-lookup"><span data-stu-id="01c01-199">When you specify multiple target frameworks, you may conditionally reference assemblies for each target framework.</span></span> <span data-ttu-id="01c01-200">En el código, puede compilar de forma condicional en esos ensamblados utilizando símbolos de preprocesador con lógica *if-then-else*.</span><span class="sxs-lookup"><span data-stu-id="01c01-200">In your code, you can conditionally compile against those assemblies by using preprocessor symbols with *if-then-else* logic.</span></span>

<span data-ttu-id="01c01-201">El siguiente archivo de proyecto de biblioteca tiene como destino las API de .NET Standard (`netstandard1.4`) y las API de .NET Framework (`net40` y `net45`).</span><span class="sxs-lookup"><span data-stu-id="01c01-201">The following library project file targets APIs of .NET Standard (`netstandard1.4`) and APIs of the .NET Framework (`net40` and `net45`).</span></span> <span data-ttu-id="01c01-202">Use el elemento **TargetFrameworks** plural con varias plataformas de destino.</span><span class="sxs-lookup"><span data-stu-id="01c01-202">Use the plural **TargetFrameworks** element with multiple target frameworks.</span></span> <span data-ttu-id="01c01-203">Tenga en cuenta cómo los atributos `Condition` incluyen paquetes específicos de la implementación cuando se compila la biblioteca para los dos TFM de .NET Framework:</span><span class="sxs-lookup"><span data-stu-id="01c01-203">Note how the `Condition` attributes include implementation-specific packages when the library is compiled for the two .NET Framework TFMs:</span></span>

```xml
  <PropertyGroup>
    <TargetFrameworks>netstandard1.4;net40;net45</TargetFrameworks>
  </PropertyGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.0 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net40' ">
    <Reference Include="System.Net" />
  </ItemGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.5 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net45' ">
    <Reference Include="System.Net.Http" />
    <Reference Include="System.Threading.Tasks" />
  </ItemGroup>

</Project>
```

<span data-ttu-id="01c01-204">Dentro de su aplicación o biblioteca, puede escribir código condicional para compilar para cada plataforma de destino:</span><span class="sxs-lookup"><span data-stu-id="01c01-204">Within your library or app, you write conditional code to compile for each target framework:</span></span>
